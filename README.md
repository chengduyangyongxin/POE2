# POE2 攻略版本库

《流放之路：降临》（Path of Exile 2）攻略资料仓库，按服务版本、赛季、玩法、职业分层管理。

## 目录结构

```
POE2/
├── global/                 # 国际服
│   ├── 新手攻略/           # 零基础、FAQ（与赛季无关）
│   └── <赛季名称>/
│       └── <玩法>/
│           └── <职业>/
│               └── （攻略文档）
└── cn-wegame/              # 国服（WeGame 版）
    ├── 新手攻略/           # 零基础、FAQ（与赛季无关）
    └── <赛季名称>/
        └── <玩法>/
            └── <职业>/
                └── （攻略文档）
```

路径规则：

- **赛季攻略**：**服务版本 / 赛季 / 玩法 / 职业**
- **新手攻略**：**服务版本 / 新手攻略**（跨赛季、跨职业的基础内容与常见问题）

**本地资料**：从网络搜集的参考草稿放在根目录 `资料/` 下，已 `.gitignore`，不提交仓库。

## 服务版本

| 目录 / 分支 | 说明 |
|-------------|------|
| `global/` · 分支 `global` | 国际服（Global） |
| `cn-wegame/` · 分支 `cn-wegame` | 国服 WeGame 版 |

国际服与国服在版本节奏、词缀、经济等方面可能存在差异，攻略请分别维护。

## 赛季

在对应服务版本目录下，以赛季名称建立文件夹，例如：

- `global/0.3-先驱赛季/`
- `cn-wegame/0.3-先驱赛季/`

新赛季开始时，可复制 `赛季模板/` 目录并重命名为新赛季名称。

## 玩法

| 玩法 | 说明 |
|------|------|
| 开荒 | 新赛季起号、剧情推进、初期装备与天赋规划 |
| 刷图 | 中后期地图刷怪、效率与收益优化 |
| Boss | 首领战机制、配装与打法 |
| 速刷 | 高效率清图、移动与技能循环 |
| 交易 | 物价参考、搬砖思路、装备买卖 |

可按实际需要增删玩法分类。

## 新手攻略

各服务版本根目录下的 `新手攻略/`，用于维护 **与赛季、职业无关** 的入门内容：

| 内容 | 说明 |
|------|------|
| 游戏入门、职业选择 | 零基础与选角 |
| 核心机制、装备、天赋 | 通用机制说明 |
| 货币、异界 | 经济与进图入门 |
| 常见问题 | FAQ 合集 |
| 版本备忘 | 当前版本特有机制（随补丁更新） |

国服索引：[`cn-wegame/新手攻略/README.md`](cn-wegame/新手攻略/README.md)

## 职业

国服与国际服职业**中文译名不同**，请按服务版本分别维护：

- **国服**：见 [`cn-wegame/README.md`](cn-wegame/README.md)（战士、游侠、女猎手、行者、女巫、魔巫、佣兵、德鲁伊）
- **国际服**：见 `global/` 下各赛季目录（本仓库暂不维护国际服职业列表）

各职业下可放置 BD 构建、技能链路、天赋、装备等攻略文档（建议使用 Markdown）。

## 分支说明

| 分支 | 用途 |
|------|------|
| `main` | 主分支，**默认拉取此分支即可看到最新国服攻略** |
| `cn-wegame` | 国服 WeGame 专属内容维护（与 `main` 同步更新） |
| `global` | 国际服专属内容维护 |

## 协作与克隆

仓库地址：<https://github.com/chengduyangyongxin/POE2>

### 获取攻略

```bash
# 推荐：克隆默认主分支（含 cn-wegame/ 下全部攻略）
git clone https://github.com/chengduyangyongxin/POE2.git

# 或显式指定国服分支
git clone -b cn-wegame https://github.com/chengduyangyongxin/POE2.git
```

进入目录后更新：

```bash
git pull
```

国服开荒入口：[`cn-wegame/0.5-奥杜尔秘符/开荒/README.md`](cn-wegame/0.5-奥杜尔秘符/开荒/README.md)  
新手入门：[`cn-wegame/新手攻略/README.md`](cn-wegame/新手攻略/README.md)

### 权限说明

| 需求 | 做法 |
|------|------|
| 只看 / clone / pull | 公开仓库，**无需**添加协作者 |
| 直接 push 到仓库 | GitHub → 仓库 **Settings** → **Collaborators** → **Add people**，邀请对方并赋予 **Write** 权限 |

若对方 `git pull` 看不到攻略，请确认是否在 **`main`** 或 **`cn-wegame`** 分支，且已执行 `git pull` 拉取最新提交。

## 文档命名建议

- 使用 Markdown（`.md`）编写攻略
- 文件名简洁明了，例如：`BD构建.md`、`开荒路线.md`、`装备选择.md`
- 同一路径下的文档按主题拆分，避免单文件过长
