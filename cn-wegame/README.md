# 国服（WeGame）目录说明

国服攻略统一放在 `cn-wegame/` 下。

| 类型 | 路径规则 |
|------|----------|
| 赛季攻略 | **赛季名称 / 玩法 / 职业** |
| 新手攻略 | **`新手攻略/`**（与赛季无关，见下方） |

新赛季可复制 `赛季模板/` 并重命名（如 `0.5-奥杜尔秘符`）。

---

## 职业目录（国服官方命名）

以 [国服新手站](https://poe2.qq.com/act/a20250228tyro/pages/page1-1.html) 为准，当前已开放 **8 个职业**（官网预留 4 个「未公开」栏位）。

| 目录名 | 主属性 | 武器 | 可选升华 |
|--------|--------|------|----------|
| `战士/` | 力量 | 锤、斧等 | 泰坦、战争使者、奇塔弗匠师 |
| `游侠/` | 敏捷 | 弓 | 锐眼、追猎者 |
| `女猎手/` | 敏捷 | 战矛 | 仪祭师、亚马逊、灵魂行者 |
| `行者/` | 敏捷/智慧 | 徒手、棍 | 祈求者、夏乌拉追随者、武圣 |
| `女巫/` | 智慧 | 法杖等 | 命源法师、驱炎使、巫妖 |
| `魔巫/` | 智慧 | 法杖 | 风暴编织者、塑时术师、瓦拉煞的门徒 |
| `佣兵/` | 力量/敏捷 | 战弩 | 猎巫人、古灵使徒斗士、战术家 |
| `德鲁伊/` | 力量/智慧 | 魔符 | 萨满、神谕者 |

## 玩法目录

| 目录名 | 说明 |
|--------|------|
| `开荒/` | 起号、剧情、初期天赋与装备 |
| `刷图/` | 中后期地图效率与收益 |
| `Boss/` | 首领机制与配装 |
| `速刷/` | 高效率清图与技能循环 |
| `交易/` | 物价、搬砖、买卖思路 |

玩法目录下的 **跨职业通用文档**（如赛季开荒须知）可放在对应玩法根目录。

**网络搜集的参考资料**（版本概览、机制摘要、链接汇总等）放在仓库根目录 **`资料/`**，已加入 `.gitignore`，不提交 Git。

## 目录示例

```
cn-wegame/
├── README.md
├── 新手攻略/
│   ├── README.md
│   ├── 01-游戏入门.md
│   └── …
├── 赛季模板/
│   └── 开荒/
│       └── 战士/
│           └── BD构建.md
└── 0.5-奥杜尔秘符/
    └── 开荒/
        ├── 佣兵/
        │   ├── README.md          # 职业机制 + BD 索引
        │   └── BD01-战术家-榴弹开荒.md
        └── 女巫/
            ├── README.md
            └── ...
```

每个职业目录下应有 **`README.md`**（职业基础、技能/天赋/装备说明、BD 索引），具体构筑写在 `BD*.md`。
