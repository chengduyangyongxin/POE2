# 02 · 职业选择

> 状态：**草稿** · 选职业前阅读

国服当前开放 **8 个职业**。完整升华与武器表见 [`cn-wegame/README.md`](../README.md)。

---

## 一、先搞懂三个概念

| 概念 | 一句话 |
|------|--------|
| **主属性** | 力量 / 敏捷 / 智慧，决定能否穿戴装备、部分技能需求 |
| **升华** | 职业「专精」，剧情中通过试炼解锁，大幅改变玩法与生存 |
| **BD** | Build，即 **技能 + 天赋 + 装备** 的整体搭配 |

同一职业可以有完全不同的 BD（例如佣兵榴弹 vs 电喷）。选职业是选大方向，具体玩法看 BD 文档。

## 三、新手友好度参考

以下为 **0.5 赛季开荒期** 主观参考，会随版本与 BD 生态变化：

| 职业 | 开荒友好度 | 简要理由 |
|------|------------|----------|
| **佣兵** | ★★★★☆ | 远程、范围伤、对黄装友好；本仓库有完整 [佣兵开荒 BD](../0.5-奥杜尔秘符/开荒/佣兵/README.md) |
| 女巫 | ★★★☆☆ | 法术开荒传统强势，但精魂与施法节奏需适应 |
| 游侠 | ★★★☆☆ | 远程安全，部分 BD 对装备要求偏高 |
| 战士 | ★★☆☆☆ | 近战需熟悉走位，生存压力大 |
| 其他 | 待补充 | 欢迎按赛季补充 BD 后更新本表 |

> **没有绝对「最强职业」**，只有适不适合你。有完整攻略的 BD 往往比「理论上强」更重要。

## 四、按游玩风格选职业

| 你想怎么玩 | 可考虑 | 备注 |
|------------|--------|------|
| 远程、少贴身 | 佣兵、游侠、女巫 | 佣兵 BD01 榴弹容错高 |
| 近战爽感 | 战士、女猎手、行者 | 建议有 ARPG 经验 |
| 召唤 / 图腾 | 女巫、德鲁伊 | 需查具体 BD |
| 低操作、高容错 | **佣兵 · 战术家榴弹** | 定身控场，见 [BD01](../0.5-奥杜尔秘符/开荒/佣兵/BD01-战术家-榴弹开荒.md) |
| 愿意管理双武器组 | 佣兵 · 电喷等 | 清图快，操作与天赋管理成本高 |

## 五、选完职业之后

1. 阅读 [03 核心机制](./03-核心机制.md)、[05 天赋与升华](./05-天赋与升华.md)
2. 进入 [`0.5-奥杜尔秘符/开荒/<职业>/`](../0.5-奥杜尔秘符/开荒/) 选择 BD
3. 版本特有提示见 [0.5 版本备忘](./版本备忘/0.5-远古回响.md)

若仓库中尚无该职业开荒文档，可暂用佣兵 BD 体验赛季节奏，或查阅 POE2DB / 社区攻略后再定。
