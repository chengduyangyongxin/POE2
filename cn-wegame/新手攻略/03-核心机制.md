# 03 · 核心机制

> 状态：**草稿** · 理解数值与战斗的基础

---

## 一、三条资源条

| 资源 | 作用 | 新手要点 |
|------|------|----------|
| **生命** | 归零则死亡（护盾先扣的情况见第四节） | 装备与天赋优先保证 **有效生命**，不要裸堆伤害 |
| **魔力** | 施放多数技能消耗魔力 | 保留回魔手段（药剂、偷取、护身符等） |
| **精魂** | 开启 **光环**、部分机制技能占用 | 前期常不够用，**不要贪开多个光环** |

开荒常见错误：伤害够但精魂不够开捷，或魔力见底放不出技能。打开角色面板（`C`）随时查看三项数值。

## 二、三大属性

| 属性 | 主要影响 | 装备上的体现 |
|------|----------|--------------|
| **力量** | 生命、**护甲** | 重甲、近战武器 |
| **敏捷** | **闪避**、部分攻速 | 轻甲、弓弩、战矛 |
| **智慧** | **魔力**、能量护盾 | 法杖、法术向装备 |

- 装备带有属性 **需求**；不够则无法穿戴。
- 辅助宝石有时也需要额外智慧，靠装备或天赋补足即可。
- 属性不够：城镇 **兜帽人** 用金币洗 **属性点**（不是天赋点），详见 [05 天赋与升华](./05-天赋与升华.md)。

## 三、四抗（元素抗性）

角色面板可见四种元素抗性：

| 抗性 | 对应伤害 |
|------|----------|
| 火焰抗性 | 火焰伤害 |
| 冰冷抗性 | 冰冷伤害 |
| 闪电抗性 | 闪电伤害 |
| 混沌抗性 | 混沌伤害（通常更难堆，上限规则与元素抗可能不同） |

### 新手必知

- 元素抗性常见 **软上限 75%**（通过装备、天赋、药剂等堆叠）。
- **负抗性** 会放大受到的伤害，换装备时注意别让抗性掉到负数。
- **穿透**：敌人可降低你的有效抗性，后期地图词缀会出现，开荒先保证正抗性。

### 各阶段建议（粗略）

| 阶段 | 目标 |
|------|------|
| 剧情前中期 | 有什么穿什么，避免单抗过低 |
| 剧情后期 / 进异界前 | 三元素抗尽量向 **60%+** 靠拢 |
| 异界稳定刷图 | 三元素抗 **75%** 为常见及格线 |

混沌抗可以循序渐进，但别长期 **-60%** 硬抗高混沌伤地图。

## 四、三种防御层

| 类型 | 机制简述 | 常见来源 |
|------|----------|----------|
| **护甲** | 降低受到的 **打击（物理命中）** 伤害 | 力量向装备、天赋；佣兵战术家常主堆护甲 |
| **闪避** | 概率 **完全躲开** 攻击 | 敏捷向装备、天赋 |
| **能量护盾** | 伤害先扣护盾，护盾有时间回复 | 智慧向装备、部分法术 BD |

### 有效生命（EHP）

不同防御方式等效于「能多扛多少伤害」。**只堆一种防御** 容易被特定伤害类型击穿；护甲流仍建议配生命与元素减伤（如战术家 **轻装与掩蔽**）。

### 混合防御

POE2 很多 BD 使用 **护甲 + 闪避 hybrid** 或 **结界类机制**（如猎巫人元素结界）。不必强求纯一种，看 BD 文档推荐。

## 五、伤害类型

| 分类 | 说明 |
|------|------|
| **物理** | 攻击命中、部分技能；与护甲互动 |
| **元素** | 火 / 冰 / 电；与对应抗性互动 |
| **混沌** | 常绕过部分常规防御，需生命或专用减伤 |

### 攻击 vs 法术

- **攻击**：依赖武器面板、攻击技能、攻速等；词缀找「附加物理/元素点伤」「攻击技能等级」。
- **法术**：依赖法术伤害%、施法速度等；词缀找「法术伤害」「法术技能等级」。

选装备时 **对齐你的主技能类型**，否则伤害词缀吃不到。

### 持续伤害（DoT）

点燃、中毒、流血等属于 **持续伤害**，与单次命中伤害分开计算。开荒了解即可，特定 BD 会深度利用。

## 六、异常状态与控制

| 状态 | 效果 | 新手提示 |
|------|------|----------|
| **晕眩** | 敌人短暂无法行动 | 闪光榴弹等技能可触发 |
| **冰冻** | 敌人无法行动 | 冰冷伤害相关 |
| **定身** | 敌人无法行动 | 佣兵 **战术家**「正中下怀」：投射物命中叠定身，对 BOSS 友好 |
| **感电** | 敌人受到更多闪电相关伤害 | 雷霆之捷、闪电协调等联动 |
| **点燃 / 中毒** | 持续伤害 | 灰烬之捷、瓦斯榴弹等 |

**控制 = 免费生存**：怪动不了就不会打你。开荒暴毙多时，优先考虑控场技能而非纯加伤害。

## 七、移动与位移

| 概念 | 说明 |
|------|------|
| **移动速度** | 鞋子常带 **+30% 及以上** 移速词缀，开荒强烈建议 |
| **翻滚** | 默认闪避动作，无敌帧短，用于微调站位 |
| **位移技能** | 如 **扑击**、闪现类；用于脱困、BOSS 走位 |
| **被围殴时** | 先控场（晕眩/定身）或位移拉开，再输出；贴脸硬扛是常见暴毙原因 |

## 八、精魂与光环

### 精魂占用

- **光环类技能**（如各种「之捷」）开启后 **持续占用精魂**，关闭才释放。
- **战旗**、部分增益同样占精魂。
- 精魂上限来自：基础值 + 天赋 + 装备（头盔、项链等常带 `+精魂`）。

### 开荒建议

| 原则 | 说明 |
|------|------|
| 先保主技能五连 | 孔位与辅助宝石优先给核心输出 |
| 捷光环 1 个起 | 如佣兵 BD01 默认 **灰烬之捷**，精魂紧张时不强开第二个 |
| 后期再加战旗 | 战术家「价高者得」与战旗联动强，但开荒期可暂缓 |

### 捷类技能

「之捷」类光环在击败敌人或造成特定伤害时触发额外效果（如点燃、感电），是 **清图联动的核心**。具体连接见各职业 BD 文档。

## 九、与 POE1 玩家的差异提示

| POE2 变化 | 说明 |
|-----------|------|
| **双武器组** | 可配置两套武器与技能，部分 BD 依赖切换（如电喷） |
| **精魂系统** | 光环占用精魂，规划与 POE1 魔力保留习惯不同 |
| **技能宝石** | 有孔位与连接规则，任务与商人供应方式变化 |
| **防御体系** | 护甲/闪避/护盾公式与 POE1 不同，勿凭感觉硬抗 |
| **升华试炼** | 获取与点法有独立流程，见 [05 天赋与升华](./05-天赋与升华.md) |
| **国服译名** | 行者、魔巫等与国际服 Monk、Sorceress 等对应，查资料时注意 |

**建议**：把 POE2 当新游戏学机制，POE1 经验作参考而非真理。
