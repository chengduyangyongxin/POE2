# BD02 · 战术家 · 电喷开荒

> 国服 0.5 奥杜尔秘符 · 清图向，可剧情后期或通关后转型  
> 难度：★★★☆☆ · 造价：中 · **必须双武器组**

---

## 一、BD 概述

**电喷流**指以 **电光散射** 清图、**轰雷弩箭** 攻坚的闪电投射物 BD。战术家 **正中下怀** 定身全场，配合 **战旗** 与 **迫击大炮** 引爆弩箭，0.5 国服社区热度高。

| 优势 | 劣势 |
|------|------|
| 清图极快，连锁 + 闪电之捷 | 需两套武器天赋，管理成本高 |
| 定身 BOSS，生存安逸 | 属性需求高（力/敏/智） |
| 攻坚爆发高（轰雷 + 迫击炮） | 纯剧情前期不如榴弹顺滑 |

**建议路径**：剧情用 [BD01 榴弹](./BD01-战术家-榴弹开荒.md) 到 **60 级进异界**，再按本文换技能与天赋；或第二幕后已拿到核心宝石时直接转型。

## 三、技能

### 3.1 双武器组分工

| 武器组 | 绑定天赋页 | 主技能 |
|--------|------------|--------|
| **红色页** | 第一套战弩 | 电光散射、轰雷弩箭、战旗 |
| **绿色页** | 第二套战弩 | 迫击大炮、高爆榴弹（引爆用） |

> 双天赋页依赖 **第二武器组** 机制：迫击炮、榴弹引爆链绑 **绿色武器天赋**；电光散射、轰雷绑 **红色武器天赋**。若尚未解锁双页，迫击炮相关大点可暂放常规树，优先保证电光散射伤害。

### 3.2 技能连接

#### 清图 · 电光散射（红页）

```
电光散射 — 阿尔琼的勋章 — 亢奋 — 闪电协调 — 分叉 — 快速攻击 II
```

- 自带连锁；配合 **闪电之捷** 清屏。
- 稀有怪伤害不足：开 **战旗** + 架 **迫击大炮**。

#### 攻坚 · 轰雷弩箭（红页）

```
轰雷弩箭 — 范围击中 — 元素军械 II — 闪电穿透 — 闪电协调 — 闪电专精
```

- BOSS 开场有减伤：战前铺满地轰雷弩箭。
- BOSS 出手前 1–2 秒放 **迫击大炮**，自动 **高爆榴弹** 引爆弩箭 → 爆发。

#### 引爆辅助 · 迫击大炮（绿页）

```
迫击大炮 — 高爆榴弹 — 紧急图腾 III — 胶粘榴弹 II — 短引线 II — 子母弹
```

#### 功能 · 紧急装填（红页）

```
紧急装填 — 冷却回复 II — 高效 II — 持续时间延长 II
```

- 刷新轰雷弩箭 CD，BOSS 前多铺一轮弩箭。

#### 增益 · 战旗（红页）

```
战旗 — 德瑞索的热情 — 范围扩大 II — 持续时间延长 II
```

- 常驻插旗；与 **价高者得** 联动。

#### 光环 · 闪电之捷（红页）

```
闪电之捷 — 闪电协调 — 元素集中 — 钙质碎片 — 活体电球 II — 元素军械 II
```

- 精魂占用高，前期可删减 **活体电球** / **钙质碎片**，先保闪电协调。

#### 召唤 · 伙伴（可选）

```
伙伴 — 罗米拉的报复 — 亡命之息 — 肉盾 II — 闪电曝露 — 过载
```

- 伙伴在引爆前给 BOSS **感电** 可显著提高伤害。

#### 位移

- **扑击** + 护身符（第二武器组）或 **翻滚** 习惯操作。

### 3.3 输出循环

**清图**

1. 常驻 **战旗** + **闪电之捷**。
2. **电光散射** 一路扫射；定身怪基本无法近身。

**BOSS**

1. 提前插 **战旗**，向脚下与 BOSS 路径铺 **轰雷弩箭**。
2. **紧急装填** 再铺一轮。
3. BOSS 攻击前放 **迫击大炮** 引爆。
4. 未秒掉则怪已被定身，继续 **电光散射** 或补弩箭。
5. 有伙伴时先让伙伴打出感电再引爆。

## 四、天赋

### 4.1 属性需求（毕业向参考）

| 属性 | 约需 | 用途 |
|------|------|------|
| 力量 | 104+ | **永恒怒火** 14 级等 |
| 敏捷 | 137+ | **闪电之捷** 精魂需求 |
| 智慧 | 40+ | 蓝色辅助孔 |

剧情阶段用 **博学**、装备补属性，不必提前硬点。

### 4.2 红色天赋页（电光 / 轰雷）

| 方向 | 节点 |
|------|------|
| 闪电 | 闪电伤害、感电、穿透、投射物速度 |
| 定身 | **恐惧麻痹**（定身 → 威吓） |
| 生存 | 生命、闪避/护甲 hybrid、元素转护甲 |
| 大点 | **风舞者**（闪避增伤）、**永恒怒火**（后期） |

### 4.3 绿色天赋页（迫击 / 榴弹引爆）

| 方向 | 节点 |
|------|------|
| 图腾 | 图腾伤害、召唤图腾上限（**炮兵就位！** 升华可选） |
| 榴弹 | 仅服务迫击炮引爆链，不必点满榴弹开荒集群 |
| 范围 | 效果区域、榴弹相关 |

### 4.4 加点原则

- **全局增伤**（闪电、投射物）放 **红色常规树**。
- **仅迫击炮受益** 的图腾/榴弹大点放 **绿色武器页**。
- 与 [BD01](./BD01-战术家-榴弹开荒.md) 相反：榴弹 BD 全局点常规树；电喷 BD 必须拆分红绿两页。

## 五、装备

### 5.1 武器

| 用途 | 基底 | 词缀 |
|------|------|------|
| 清图弩 | 炮击/攻城战弩 | 闪电点伤、投射物速度、+技能等级 |
| 攻坚弩 | 同上或高暴击 | 轰雷弩箭等级、闪电伤、暴击 |

两把弩可分工，或一把过渡用到底。

### 5.2 防具

- 电喷偏 **闪避 + 护甲 hybrid**（与榴弹纯护甲不同，但仍可吃战术家升华）。
- 四抗满为前提；追求稀有度时单件 **物品稀有度** 约 100% 上限，需牺牲防御。
- 手套/戒指：闪电点伤、攻击速度、生命。

### 5.3 饰品与药剂

| 类型 | 推荐 |
|------|------|
| 项链 | +闪电技能等级、生命、暴击（轰雷向） |
| 戒指 | 闪电伤、生命、抗性 |
| 护符 | 弩伤提高、伙伴伤害（若带伙伴） |
| 药剂 | 血药瞬间回复；蓝药高回复 |

### 5.4 可选暗金

| 暗金 | 说明 |
|------|------|
| **三龙** | 若改用火焰捷过渡可用；电喷主玩闪电之捷非刚需 |
| 高闪避头盔/胸 | 抬 **元素结界** 前先堆闪避（若副修猎巫人线） |

### 5.5 0.5 符文结界

- 清图站撸多：给胸甲打 **符文结界**（≤55 级无惩罚）。
- 高闪避流结界作为「第二条命」。
