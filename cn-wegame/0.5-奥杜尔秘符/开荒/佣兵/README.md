# 佣兵 · 0.5 开荒攻略索引

> 国服 · 奥杜尔秘符赛季 · 开荒向  
> 技能译名以 [POE2DB 中文](https://poe2db.tw/cn/) 为准

---

## 职业基础

| 项目 | 说明 |
|------|------|
| 主属性 | **力量** + **敏捷** |
| 主武器 | **战弩**（榴弹、弩箭技能均依赖战弩） |
| 防御倾向 | **护甲** + **闪避** hybrid |
| 职业特性 | **图腾持续时间延长** |
| 可选升华 | **战术家**、**猎巫人**、**古灵使徒斗士** |

佣兵开荒优势：远程、范围伤害高、对自制装备友好、剧情 BOSS 爆发强。劣势：榴弹类技能 **屏幕特效多**，部分玩家可能晕屏或掉帧。

---

## 技能体系

### 武器与技能类型

- 几乎所有输出技能绑定 **战弩**，换技能不换武器类型。
- 两大分支：**榴弹系**（高爆榴弹、闪光榴弹、瓦斯榴弹）与 **弩箭系**（电光散射、轰雷弩箭等）。
- 榴弹有 **引线时间**，需预判；弩箭偏 **攻速 + 连锁**。

### 精魂与光环

- **捷光环**（灰烬之捷、雷霆之捷等）与 **战旗** 占精魂，开荒先保主技能五连。
- 战术家 **战旗** 与「价高者得」升华联动，后期再完善。
- 猎巫人 **元素结界** 是主动技能，不占精魂但需操作。

### 技能连接要点

| 概念 | 作用 |
|------|------|
| **高爆榴弹** | 主范围物理/火焰爆发 |
| **闪光榴弹** | 晕眩控场、脱困 |
| **瓦斯榴弹** | 铺毒/减抗，配合引爆链 |
| **迫击大炮 + 引爆** | 异界清图核心组合 |
| **定身**（战术家） | 投射物命中叠定身，BOSS 无法行动 |
| **图腾**（部分 BD） | 脱手引爆/迫击，吃职业图腾持续加成 |

### 双武器组

- **BD02 电喷** 必须双武器组：绿页迫击/榴弹 + 红页电光散射/轰雷。
- 剧情通用：**护身符 + 扑击 + 虹吸印记** 绑第二武器组，BOSS 回蓝。

## 天赋与升华

### 天赋树特点

- 力量/敏捷交界，点 **护甲、生命、投射物、榴弹、攻速**。
- 战术家开荒堆 **护甲% 转元素减伤**（轻装与掩蔽）收益极高。
- 部分辅助宝石需 **智慧**，装备补少量即可。

### 三大升华方向

| 升华 | 定位 | 核心机制 |
|------|------|----------|
| **战术家** | 定身 + 战旗 | 正中下怀定身、价高者得战旗、轻装与掩蔽 |
| **猎巫人** | 结界坦 | 元素结界、终结、护甲闪避双堆 |
| **古灵使徒斗士** | 珠宝镶嵌 | 血肉嵌珠宝、超多技能槽，构筑复杂 |

**开荒建议**：纯开荒 **战术家 BD01 榴弹**；剧情后转清图 **BD02 电喷**；常死试 **BD03 猎巫人**。

## 装备体系

### 防御思路

- **护甲** 是战术家核心；猎巫人 **护甲 + 闪避** 双堆抬 **元素结界** 上限。
- 词条「**护甲值的 X% 也适用于元素伤害**」与战术家升华 synergy 极强。
- 0.5：**符文结界** 55 级以下无惩罚，剧情积极做维金。

### 武器与词缀

| 阶段 | 优先 |
|------|------|
| 剧情 | 战弩 **物理/火焰点伤** 面板最高即可 |
| 过渡 | +榴弹/投射物技能等级、攻速 |
| 异界 | 生命、护甲、四抗；暗金三龙（火焰转感电）等 |

### 防具与饰品

| 部位 | 优先 |
|------|------|
| 胸甲/头 | 高护甲或护甲/闪避 hybrid、生命 |
| 手套 | 皮缚护手（钢铁意志转护甲）、点伤 |
| 鞋 | **30%+ 移速**、生命 |
| 戒指/项链 | 生命、抗性、物理/火焰点伤 |

### 剧情任务奖励

见各 BD「**六、剧情任务奖励（建议）**」：技能宝石、战弩、抗性。

## 开荒注意事项

1. 榴弹 **引线 + 定身** 节奏需练习，勿贴脸硬炸。
2. 战术家定身强但需 **投射物命中**，距离过远有轻装与掩蔽减命中补偿。
3. 电喷转型需 **双天赋页**，洗点与第二武器组都要准备好。
4. 猎巫人开 **元素结界** 前确保甲闪总和够高（约 2 万+ 有可观屏障）。
5. 特效敏感玩家优先调低粒子或选弩箭流。

## 通用开荒提示（国服）

1. 技能宝石：每升一级检查任务奖励与商人；**剧情装备奖励**见各 BD 第六节。
2. 装备：优先 **生命、四抗、移动速度（鞋 30%+）**；战术家额外堆 **护甲**。
3. 0.5 赛季：尽早做法罗任务，低等级护甲加 **符文结界**（55 级以下无惩罚）。
4. 属性不够：城镇 **兜帽人** 或 **多里亚尼** 洗点。
5. 国服组队掉落削弱，独狼或固定队往往比野队 6 人更划算。
