## 职业基础

| 项目 | 说明 |
|------|------|
| 主属性 | **力量** + **敏捷** |
| 主武器 | **战弩**（榴弹、弩箭技能均依赖战弩） |
| 防御倾向 | **护甲** + **闪避** hybrid |
| 职业特性 | **图腾持续时间延长** |
| 可选升华 | **战术家**、**猎巫人**、**古灵使徒斗士** |

远程输出；主武器为 **战弩**。技能分支含榴弹系与弩箭系；榴弹技能屏幕特效较多。

---

## 技能体系

### 武器与技能类型

- 几乎所有输出技能绑定 **战弩**，换技能不换武器类型。
- 两大分支：**榴弹系**（高爆榴弹、闪光榴弹、瓦斯榴弹）与 **弩箭系**（电光散射、轰雷弩箭等）。
- 榴弹有 **引线时间**，需预判；弩箭偏 **攻速 + 连锁**。

### 精魂与光环

- **捷光环**（灰烬之捷、雷霆之捷等）与 **战旗** 占精魂；主技能五连所需精魂通常高于捷光环等常驻技能。
- 战术家 **战旗** 与「价高者得」升华联动，后期再完善。
- 猎巫人 **元素结界** 是主动技能，不占精魂但需操作。

### 技能连接要点

| 概念 | 作用 |
|------|------|
| **高爆榴弹** | 主范围物理/火焰爆发 |
| **闪光榴弹** | 晕眩控场、脱困 |
| **瓦斯榴弹** | 铺毒/减抗，配合引爆链 |
| **迫击大炮 + 引爆** | 异界清图核心组合 |
| **定身**（战术家） | 投射物命中叠定身，BOSS 无法行动 |
| **图腾**（部分 BD） | 脱手引爆/迫击，吃职业图腾持续加成 |

### 双武器组

- **BD02 电喷** 必须双武器组：绿页迫击/榴弹 + 红页电光散射/轰雷。
- 剧情通用：**护身符 + 扑击 + 虹吸印记** 绑第二武器组，BOSS 回蓝。

## 天赋与升华

### 天赋树特点

- 力量/敏捷交界，点 **护甲、生命、投射物、榴弹、攻速**。
- 战术家天赋中 **护甲% 转元素减伤**（轻装与掩蔽）与护甲词条联动。
- 部分辅助宝石需 **智慧**，装备补少量即可。

### 三大升华方向

| 升华 | 玩法类型 | 核心机制 |
|------|------|----------|
| **战术家** | 定身 + 战旗 | 正中下怀定身、价高者得战旗、轻装与掩蔽 |
| **猎巫人** | 结界坦 | 元素结界、终结、护甲闪避双堆 |
| **古灵使徒斗士** | 珠宝镶嵌 | 血肉嵌珠宝、扩展技能槽 |

## 装备体系

### 防御思路

- **护甲** 是战术家核心；猎巫人 **护甲 + 闪避** 双堆抬 **元素结界** 上限。
- 词条「**护甲值的 X% 也适用于元素伤害**」与战术家升华「轻装与掩蔽」联动。
- 0.5：**符文结界** 在 55 级以下无等级惩罚；剧情阶段可使用维金附加。

### 武器与词缀

| 阶段 | 常见词缀 |
|------|------|
| 剧情 | 战弩 **物理/火焰点伤** 面板最高即可 |
| 过渡 | +榴弹/投射物技能等级、攻速 |
| 异界 | 生命、护甲、四抗；暗金三龙（火焰转感电）等 |

### 防具与饰品

| 部位 | 常见词缀 |
|------|------|
| 胸甲/头 | 高护甲或护甲/闪避 hybrid、生命 |
| 手套 | 皮缚护手（钢铁意志转护甲）、点伤 |
| 鞋 | **30%+ 移速**、生命 |
| 戒指/项链 | 生命、抗性、物理/火焰点伤 |

### 剧情任务奖励

见各 BD「**六、剧情任务奖励**」：技能宝石、战弩、抗性。

## 机制说明

1. 榴弹技能含 **引线时间** 与 **定身** 机制；近距引爆时角色位于爆炸范围内。
2. 战术家定身依赖 **投射物命中**；距离过远时「轻装与掩蔽」提供命中补偿。
3. 电喷构筑需 **双天赋页** 与第二武器组。
4. 猎巫人 **元素结界** 屏障上限与护甲、闪避总和相关（约 2 万+ 甲闪总和时有可观屏障）。
5. 榴弹类技能屏幕特效较多；可调整粒子设置或选用弩箭系技能。

详见 [新手攻略 · 常见问题](../../../新手攻略/常见问题.md) 与 [0.5 版本备忘](../../../新手攻略/版本备忘/0.5-远古回响.md)。摘要：
