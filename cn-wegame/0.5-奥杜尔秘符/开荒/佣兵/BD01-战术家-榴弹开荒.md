# BD01 · 战术家 · 榴弹开荒

> 国服 0.5 奥杜尔秘符 · 剧情至异界初期  
> 难度：★☆☆☆☆ · 造价：低 · 暗金依赖：可选

---

## 一、BD 概述

以 **高爆榴弹** 为核心输出的远程爆破流，配合 **闪光榴弹** 控场、**瓦斯榴弹** 铺毒/减抗，战术家 **定身** 提供顶级容错。不依赖昂贵暗金，国际服与国服社区均验证为 **T0 级开荒方案**。

| 优势 | 劣势 |
|------|------|
| BOSS 爆发高、清图范围大 | 爆炸特效多，可能晕屏 |
| 定身怪无法攻击你 | 榴弹有引线，需预判走位 |
| 自制装备即可成型 | 后期想极限效率需换电喷或投资 |

---

## 二、升华：战术家

### 试炼获取顺序（建议）

完成 **混沌试炼** 与 **sekhema 试炼** 后，按序点升华（共 8 点，以下为优先级）：

| 顺序 | 升华节点 | 作用 |
|------|----------|------|
| 1 | 定身积蓄（小点）×2 | 加快定身上线 |
| 2 | **正中下怀** | 投射物伤害产生定身；定身敌人无法行动 |
| 3 | 护甲和闪避（小点）×2 | 生存基础 |
| 4 | **价高者得** | 旗帜获荣耀、可插多面战旗 |
| 5 | **可靠的计划** | 永久性增益保留总降 50% |
| 6 | **轻装与掩蔽** | 200% 护甲防御元素伤；敌人距离越远命中越低 |

> 若剧情阶段暴毙频繁，可将 **轻装与掩蔽** 提前到第 4 步，暂缓 **价高者得**。

### 核心机制

- **定身**：比晕眩/冰冻更易对 BOSS 连续触发（无类似内置 CD），配合榴弹持续轰炸。
- **战旗**：后期可选；开荒期精魂紧张可暂不插战旗，先保证榴弹与捷光环。

## 三、技能

### 3.1 剧情阶段学习顺序

1. **高爆榴弹**（主输出）
2. **闪光榴弹**（晕眩、脱困）
3. **灰烬之捷**（精魂光环，清图联动）
4. **瓦斯榴弹**（BOSS 铺垫、毒云）
5. **爆炸射击**（可选，引爆地面榴弹，手感更顺）
6. **扑击**（第二武器组 + 护身符，位移与印记）
7. **集束榴弹** + **迫击大炮**（异界前后，阵地战、高密度怪）

### 3.2 技能连接

#### 主输出 · 高爆榴弹

```
高爆榴弹 — 多重射击 II — 元素军械 II — 易怒 II — 短引线 II — 审议
```

| 辅助 | 说明 |
|------|------|
| 多重射击 II | 榴弹第一优先；显著扩大范围与命中 |
| 元素军械 II | 元素伤害与触发捷 |
| 易怒 II | 总降引爆时间约 80% |
| 短引线 II | 与易怒配合，引爆约 **1 秒**，契合边扔边滚 |
| 审议 | 五连后稳定增伤；高压图可换 **火焰穿透 II** |

前期四连：`高爆榴弹 — 多重射击 II — 元素军械 II — 短引线 II`

#### 控场 · 闪光榴弹

```
闪光榴弹 — 多重射击 II — 晕眩 III — 审议 — 易怒 I
```

- 被包围时先扔闪光，再补高爆/集束。
- 无闪电点伤时，审议可换 **闪电协调** 打感电。

#### 铺垫 · 瓦斯榴弹

```
瓦斯榴弹 — 多重射击 II — 元素军械 II — 短引线 II — 火焰穿透 II
```

- BOSS 战：先铺瓦斯毒云，再高爆榴弹引爆，伤害显著提高。
- 战术家定身 + 火焰穿透时，**油榴弹** 非必须（省一个键位）。

#### 阵地 · 迫击大炮（16 级+ 再五连）

```
迫击大炮 — 集束榴弹 — 多重射击 II — 易怒 II — 承载 — 恢复 III
```

- 深渊、裂隙等需站桩的玩法先架炮。
- **承载** 与 **恢复 III** 成对使用，并频繁刷新迫击炮重置冷却。

#### 精魂技能

| 技能 | 连接 | 用途 |
|------|------|------|
| 灰烬之捷 | 清晰 II / 食人魔 II（可选） | 清图点燃联动 |
| 雷霆之捷 | 同上 | 配合暗金 **三龙** 头盔打感电（见装备） |
| 扑击 | 任意护身符 + **虹吸印记 II** | 第二武器组自动切换；BOSS 回蓝与吸血 |

> 开荒默认 **灰烬之捷**；捡到 **三龙** 后改 **雷霆之捷** + 火焰伤害转感电。

#### 可选 · 爆炸射击

```
爆炸射击 — 多重射击 II — 快速攻击 II — 元素军械 II — 双管齐下 II
```

- 可瞬间引爆地面榴弹，弥补引线等待；非必须，习惯榴弹节奏可不学。

### 3.3 输出循环

**清图**

1. 边走边扔 **高爆榴弹**。
2. 精英/危险怪：**闪光榴弹** → **集束榴弹**。
3. 长时间战斗（仪式等）：提前架 **迫击大炮**。

**BOSS**

1. 远距离铺 **瓦斯榴弹**。
2. **闪光榴弹** 叠晕眩/定身。
3. 倾泻 **高爆榴弹** + **集束榴弹**。
4. 冷却中走位，用 **扑击** 上印记回蓝。
5. BOSS 开场前可预扔榴弹利用其废话时间叠弹。

## 四、天赋

### 4.1 加点思路

围绕：**榴弹数量、引爆速度、冷却回复、火焰/物理投射物伤害、移动速度**；剧情阶段偏输出，进异界后补 **护甲、生命、元素转护甲**。

### 4.2 优先大点与集群

| 类别 | 推荐节点方向 |
|------|----------------|
| 榴弹 | 额外榴弹投射物、引爆时间降低、冷却回复 |
| 伤害 | 投射物伤害、火焰伤害、范围、**爆破师**（与火焰联动） |
| 定身联动 | 点出 **正中下怀** 后 → **恐惧麻痹**（定身时威吓增伤） |
| 破甲 | **护甲破碎** 集群 + 暗金 **熔火者之赐**（破甲也增火焰承受） |
| 生存 | 生命、**博学**（力量转生命）、**轻装与掩蔽** 升华配合的护甲%转元素减伤 |
| 吸血 | **快速代谢** + 闪光榴弹连 **生命偷取 III**、**穿刺** |
| 属性 | **博学** 降低装备需求；力敏智只点到能穿装备、插宝石 |

### 4.3 双武器组（异界后）

- 第一套：战弩 + 榴弹技能树。
- 第二套：护身符 + **扑击**（武器自动切换，无需手动配武器）。
- 猎巫人/电喷才重度依赖双天赋页；战术家榴弹 **单天赋树即可打完剧情**，异界再考虑细分。

### 4.4 等级阶段参考

| 阶段 | 等级 | 重点 |
|------|------|------|
| 第一幕 | 1–15 | 榴弹集群、生命、力敏 |
| 第二幕 | 16–35 | 引爆速度、多重相关、四抗 |
| 第三幕 | 36–50 | 火焰/投射物、护甲小点 |
| 进异界 | 60+ | 补护甲、恐惧麻痹、破甲联动 |

> 天赋无法贴图时，请在 POE2 国服客户端对照「榴弹」「投射物」「护甲」关键字路径点选；社区 Maxroll / 游民星空有交互天赋树可导入。

## 五、装备

### 5.1 武器

| 阶段 | 要求 |
|------|------|
| 剧情 | 物理 + 火焰 **点伤/增伤** 高的战弩即可 |
| 过渡 | **攻城战弩 / 炮击战弩** 基底，+榴弹或投射物技能等级 |
| 属性 | 攻击速度次要；优先面板伤害 |

### 5.2 防具

| 部位 | 优先级 |
|------|--------|
| 头盔 | 生命 + 抗性；可选暗金 **三龙**（火焰转感电，配雷霆之捷） |
| 胸甲 | **纯护甲** 或护甲/闪避 hybrid，高生命 |
| 手套 | 护甲基底 **皮缚护手**（配合 **钢铁意志** 转护甲）；补火/物理点伤 |
| 鞋子 | **30%+ 移动速度**、生命、抗性 |
| 腰带 | 生命、抗性、药剂充能 |

**防御核心**：战术家堆 **护甲**；词条「**护甲值的 X% 也适用于元素伤害**」越多越好，与升华 **轻装与掩蔽**  synergy 极强。

### 5.3 饰品

- 戒指：生命、抗性、物理/火焰 **附加点伤**。
- 项链：生命、技能等级（榴弹/投射物）、抗性。
- 药剂：瞬间回复生命；魔力药剂选高回复。

### 5.4 开荒暗金（可选）

| 暗金 | 作用 |
|------|------|
| **三龙** | 火焰伤转感电 → 激活雷霆之捷，增伤明显 |
| **熔火者之赐** | 破甲联动火焰伤害 |
| **愤怒的诞生**（鞋） | 油/火点燃联动；偏猎巫人油榴弹玩法，战术家非刚需 |

### 5.5 0.5 赛季机制

- 55 级以下护甲：找法罗做 **维金符文锻造** 加 **符文结界**，无防御惩罚。
- 遗物遭遇战优先做能用的护甲/武器，不必强求毕业词缀。

## 七、异界过渡

1. 补全 **集束榴弹 + 迫击大炮** 五连。
2. 天赋转向：护甲、元素减伤、恐惧麻痹完整链路。
3. 精魂够用时加 **战旗**（德瑞索的热情 + 范围 + 持续时间）。
4. 若清图仍嫌慢、且愿意管理双武器组 → 参考 [BD02 电喷](./BD02-战术家-电喷开荒.md) 转型。
