# BD03 · 猎巫人 · 榴弹开荒

> 国服 0.5 奥杜尔秘符 · 榴弹流变体  
> 难度：★★☆☆☆ · 造价：低 · 生存优于战术家

---

## 一、BD 概述

技能轴与 [BD01 战术家榴弹](./BD01-战术家-榴弹开荒.md) **基本相同**，差异在 **猎巫人升华**：提供 **终结（斩杀）**、**元素结界** 屏障、击杀爆炸等，适合容易暴毙或想体验猎巫人的玩家。

| 对比 BD01 | 说明 |
|-----------|------|
| 伤害 | 略低（无定身），但 **终结** 补刀强 |
| 生存 | **元素结界** 吸收元素伤，中期很肉 |
| 清图 | 榴弹机制一致，速度接近 |
| 后期 | 可转 **武器大师** 双天赋，兼容电喷或爆炸矢 |

---

## 二、升华：猎巫人

### 推荐加点顺序

| 顺序 | 节点 | 作用 |
|------|------|------|
| 1 | **冷血执行者** | 终结：低血敌人直接斩杀 |
| 2 | 冷却回复（小点）×2 | 榴弹循环更顺 |
| 3 | **痴狂仪式** | 获得 **元素结界**；护甲闪避总降 50% |
| 4 | **净罪礼仪** | 结界也吸收物理/混沌（需先痴狂仪式） |
| 5 | **狂热审讯** | 击杀 10% 几率爆炸，清图加速 |
| 6 | 效果区域 | 榴弹范围 |
| 7 | **武器大师** | 100 点天赋变为武器组天赋（异界后点） |

> 剧情经常死：把 **痴狂仪式** 提前到第 2 步；结界数值 = 护甲+闪避总和的 35%，堆到 2 万甲闪即可有可观屏障。

### 核心机制

- **元素结界**：替代战术家定身成为生存核心；元素伤先扣结界。
- **专注粉碎**（女巫克星线）：对法系怪额外收益，开荒次要。
- **终结**：对残血小怪极爽，BOSS 收益有限。

## 三、技能

### 3.1 与 BD01 的差异

| 项目 | BD01 战术家 | BD03 猎巫人 |
|------|-------------|-------------|
| 油榴弹 | 可不学（火焰穿透替代） | 可选；配合 **愤怒的诞生** 鞋全屏点燃 |
| 精魂防御 | 战旗 + 护甲 | **元素结界** 主动开 |
| 扑击/印记 | 推荐 | 同样推荐 |

### 3.2 技能连接（直接沿用 BD01）

**高爆榴弹**

```
高爆榴弹 — 多重射击 II — 元素军械 II — 易怒 II — 短引线 II — 审议
```

**闪光榴弹**

```
闪光榴弹 — 多重射击 II — 晕眩 III — 审议 — 易怒 I
```

**瓦斯榴弹**

```
瓦斯榴弹 — 多重射击 II — 元素军械 II — 短引线 II — 火焰穿透 II
```

**迫击大炮**（异界）

```
迫击大炮 — 集束榴弹 — 多重射击 II — 易怒 II — 承载 — 恢复 III
```

**精魂**

```
灰烬之捷 — 清晰 II
元素结界 — （升华自带，战斗中手动开）
```

### 3.3 猎巫人专属循环技巧

**油榴弹变体**（有 **愤怒的诞生** 鞋后）

1. **油榴弹** 铺地（减速 + 曝露）。
2. **瓦斯榴弹** + **高爆榴弹** 引爆油面，全屏燃烧。
3. 走路即可清图，少按爆炸射击。

无暗金鞋时，循环与 BD01 完全一致。

## 四、天赋

### 4.1 加点思路

与 BD01 相同主轴：**榴弹数量、引爆速度、冷却、投射物/火焰伤害**。

额外优先：

| 节点类型 | 说明 |
|----------|------|
| 护甲 + 闪避 | 同时抬 **元素结界** 上限 |
| 冷却回复 | 配合猎巫人小点，榴弹更密 |
| 生命 | 结界破后仍有容错 |

### 4.2 武器大师后（异界）

- 参考 [BD02](./BD02-战术家-电喷开荒.md)：一套榴弹、一套爆炸矢/弩箭。
- 榴弹全局增伤点 **常规树**；爆炸矢专属点 **武器页**。

### 4.3 剧情阶段

- 80 点前可 **主点输出**，与游民攻略一致；进异界再补甲闪集群。
- 不必过早点 **武器大师**（需较多天赋点）。

## 五、装备

### 5.1 与 BD01 的异同

| 部位 | BD01 战术家 | BD03 猎巫人 |
|------|-------------|-------------|
| 胸甲/头 | 偏 **纯护甲** | **护甲 + 闪避** 双堆，抬结界 |
| 手套 | 皮缚护手 + 钢铁意志 | 同样适用 |
| 鞋子 | 移速优先 | 可选 **愤怒的诞生**（油流） |
| 头盔 | 三龙 | 三龙 / **阿兹里的蔑视**（生存向，后期） |

### 5.2 武器

- 与 BD01 相同：高物理火焰面板战弩 → 攻城/炮击基底过渡。

### 5.3 药剂

- 生命药剂：瞬间回复 + 护甲/闪避期间可用词条。
- 魔力：高回复；猎巫人印记流依赖扑击回蓝。

### 5.4 0.5 赛季

- 符文结界与 BD01 相同，55 级以下护甲优先做结界。

## 六、升华选择建议

| 选战术家 BD01 | 选猎巫人 BD03 |
|---------------|---------------|
| 想要定身控 BOSS | 更信屏障与终结 |
| 不爱开主动防御技能 | 愿意开 **元素结界** |
| 计划纯榴弹到中期 | 计划后期 **武器大师** 多玩法 |
| 组队当辅助（价高者得） | 独狼自坦 |

---

## 七、输出循环摘要

与 BD01 相同：

- **清图**：高爆榴弹为主，闪光脱困，灰烬之捷联动。
- **BOSS**：瓦斯 → 高爆 → 闪光控场 → 集束/迫击。
- **额外**：残血怪靠 **终结** 秒；元素伤高图先开 **元素结界**。
