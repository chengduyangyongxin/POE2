## 一、BD 概述

以 **轰动猛击** + **碎骨** 动画取消为核心，泰坦 **巨躯、地裂者** 提供高额近战爆发。清图与 BOSS 速度优于盾墙流，容错略低。

| 优势 | 劣势 |
|------|------|
| 伤害高、剧情推进快 | 需贴脸，护甲要堆 |
| 晕眩链控 BOSS | 动画取消有学习成本 |
| 双手锤装备好找 | 元素伤需注意抗性 |

## 三、技能

### 3.1 学习顺序

1. **轰动猛击**（主输出）
2. **碎骨**（单体、动画取消）
3. **跃击** / **陨坑**（位移）
4. **地震** 或 **完美打击**（BOSS）
5. **战争之拳** 辅助链

### 3.2 技能连接

```
轰动猛击 — 翻涌 I — 战争之拳 I — 审议
```

```
碎骨 — 效能 I — 快速攻击 I（BOSS 切换）
```

### 3.3 循环

- 清图：轰动猛击 + 碎骨取消复读。
- BOSS：碎骨 / 完美打击；晕眩窗口砸地震。
