# BD01 · 奇塔弗匠师 · 盾墙开荒

> 国服 0.5 奥杜尔秘符 · 剧情至异界初期  
> 难度：★☆☆☆☆ · 造价：低 · 暗金依赖：几乎无

---

## 一、BD 概述

以 **盾墙** 为核心，配合 **战吼** 引爆与 **炼狱战吼** 清图，奇塔弗匠师 **火抗转全元素抗** 使剧情极易凑满抗性。社区公认的战士 **最稳开荒** 方案之一。

| 优势 | 劣势 |
|------|------|
| 护甲 + 格挡 + 盾墙，极难暴毙 | 清图速度不如泰坦猛击 |
| 剧情抗性压力小 | 需习惯战吼引爆节奏 |
| 自制盾牌即可成型 | BOSS 战偏磨 |

---

## 二、升华：奇塔弗匠师

### 试炼获取顺序（建议）

| 顺序 | 升华节点 | 作用 |
|------|----------|------|
| 1 | 锻造/护甲小点 | 生存与装备需求 |
| 2 | **火抗转元素抗** 核心节点 | 剧情凑四抗 |
| 3 | 生命/护甲小点 ×2 | 撑盾墙伤害基数 |
| 4 | **奇塔弗之力** 相关大点 | 伤害与锻造联动 |
| 5+ | 武器品质、破甲 | 异界再补 |

### 核心机制

- **盾墙**：伤害与 **盾牌护甲** 挂钩，选高护甲盾。
- **战吼**：强化自身、削弱敌人；**炼狱战吼** 引爆 corpses 清小怪。
- 22 级左右拿到盾墙后伤害质变，此前用 **轰动猛击** 过渡。

## 三、技能

### 3.1 剧情学习顺序

| 阶段 | 技能 | 说明 |
|------|------|------|
| 1 级 | **轰动猛击** | 过渡主输出至 ~22 级 |
| 6 级左右 | **碎骨** | 单体补充 |
| ~22 级 | **盾墙** | 主技能 |
| 第二幕 | **坚韧战吼** / **炼狱战吼** | 增益 + 引爆 |
| 第三幕 | **强化战吼** | 战吼体系完善 |
| 中后期 | **跃击** | 位移 |

**升级优先级**：盾墙 > 炼狱战吼 > 坚韧战吼

### 3.2 技能连接

#### 主输出 · 盾墙

```
盾墙 — 范围扩大 I — 集中效应 I — 审议
```

精魂够时加 **强化战吼** 或 **炼狱战吼** 独立施放。

#### 过渡（盾墙前）

```
轰动猛击 — 翻涌 I — 战争之拳 I
```

#### 战吼

```
炼狱战吼 — 范围扩大 I
坚韧战吼 — 效能 I（独立开）
```

### 3.3 输出循环

**清图**

1. **坚韧战吼** 起手。
2. **盾墙** 挥击清怪；尸体多时用 **炼狱战吼** 引爆。

**BOSS**

1. 坚韧战吼 → 盾墙连挥。
2. 炼狱战吼补范围；缺蓝用跃击调整站位。

## 四、天赋

### 4.1 加点思路

- **护甲、生命、格挡、战吼效果**
- 奇塔弗线：**元素抗性** 小点优先
- 剧情偏生存，异界补 **破甲、物理伤害**

### 4.2 等级阶段

| 阶段 | 重点 |
|------|------|
| 1–22 级 | 猛击过渡、生命、护甲 |
| 22–50 | 盾墙伤害、战吼、四抗 |
| 异界 | 破甲、格挡回复、符文结界 |

## 五、装备

### 5.1 武器与盾牌

| 部位 | 优先 |
|------|------|
| 盾牌 | **高护甲** 基底 > 一切（盾墙伤害来源） |
| 主手 | 锤/斧，补物理点伤 |

### 5.2 防具

- 胸甲/头：**纯护甲** + 生命
- 鞋：**30%+ 移速**、生命、抗性
- 手套：攻速、物理点伤

### 5.3 0.5 赛季机制

- ≤55 级护甲做 **符文结界**，与护甲流 synergy 强。
