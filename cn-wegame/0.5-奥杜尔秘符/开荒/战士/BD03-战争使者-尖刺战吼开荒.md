# BD03 · 战争使者 · 尖刺战吼开荒

> 国服 0.5 奥杜尔秘符 · 图腾/战吼脱手向  
> 难度：★★★☆☆ · 造价：低

---

## 一、BD 概述

**尖刺战吼** 或 **震波图腾** 脱手输出，战争使者 **战号之吼、木墙** 强化战吼与生存。适合不想持续贴脸的玩家。

| 优势 | 劣势 |
|------|------|
| 图腾/战吼脱手 | 单体 BOSS 略慢 |
| 战争使者尸体爆炸清图 | 需管理图腾位置 |
| 护甲坦度尚可 | 精魂与图腾天赋投资 |

---

## 二、升华：战争使者

| 顺序 | 升华节点 | 作用 |
|------|----------|------|
| 1 | **战号之吼** | 战吼范围与效果 |
| 2 | **木墙** | 召唤物/图腾生存 |
| 3 | 战吼伤害小点 | 输出 |
| 4 | 尸体爆炸相关 | 清图 |

## 三、技能

### 方案 A：尖刺战吼

```
尖刺战吼 — 翻涌 I — 战争之拳 I — 审议
炼狱战吼 — 范围扩大 I
```

### 方案 B：震波图腾

```
震波图腾 — 快速攻击 I — 近战物理伤害 I
先祖战士图腾（BOSS）
```

### 循环

- 清图：插图腾 → 战吼 → 走位等震波。
- BOSS：先祖战士图腾 + 尖刺战吼。
