# 战士 · 0.5 开荒攻略索引

> 国服 · 奥杜尔秘符赛季 · 开荒向  
> 技能译名以 [POE2DB 中文](https://poe2db.tw/cn/) 为准

---

## 职业基础

| 项目 | 说明 |
|------|------|
| 主属性 | **力量** |
| 主武器 | **锤、斧、巨剑** 等近战双手/单手武器 |
| 防御倾向 | **护甲** + **生命**（全职业基础生命偏高） |
| 职业特性 | 攻击命中敌人时可将其 **击退** |
| 可选升华 | **泰坦**、**战争使者**、**奇塔弗匠师** |

近战职业；主属性 **力量**；防御以 **护甲** 与生命为主；机动性低于远程职业。

---

## 技能体系

### 武器与技能类型

- 多数近战技能绑定 **力量武器**（锤/斧/剑），选 BD 前先确认技能要求的武器类型。
- 常见开荒方向：**猛击类**（泰坦）、**战吼 + 先祖**（战争使者）、**锻造/破甲**（奇塔弗匠师）。
- 部分技能可造成 **晕眩、破甲**，与战士高护甲形成「控住再打」的节奏。

### 精魂与光环

- 战士开荒精魂压力通常 **低于** 女巫/佣兵捷光环流，但战吼、先祖、部分增益仍占精魂。
- 主输出技能五连所需精魂通常高于捷光环、战吼等常驻技能。

### 技能连接要点

| 概念 | 作用 |
|------|------|
| **猛击 / 重击** | 单体 BOSS 爆发，常配合晕眩链 |
| **战吼** | 范围削弱敌人或强化自身，战争使者核心 |
| **先祖之灵** | 额外伤害来源，可脱手输出 |
| **破甲** | 降低敌人护甲，团队与 solo 均有效 |

## 天赋与升华

### 天赋树特点

- 起点在力量侧，易点到 **生命、护甲、近战伤害、晕眩**。
- 开荒期常见天赋投资顺序：**生命** → **四抗** → 输出大点；异界阶段可补护甲%、元素减伤。
- 力量够穿装备即可，多余点数可转生命或伤害。

### 三大升华方向

| 升华 | 玩法类型 | 核心机制 |
|------|------|----------|
| **泰坦** | 护甲猛击 | 晕眩、猛击强化、高护甲减伤、先祖神力 |
| **战争使者** | 战吼 + 范围爆发 | 先祖之灵、战吼使尸体爆炸、前线压制 |
| **奇塔弗匠师** | 锻造 + 破甲 | 自带锻造、武器品质、奇塔弗黑暗之力 |

---

## 装备体系

### 防御思路

- **护甲** 是主防御层；胸甲、头盔常见高护甲基底 + 生命词缀。
- 词条「**护甲值的 X% 也适用于元素伤害**」可将护甲百分比应用于元素伤害。
- 0.5 赛季：≤55 级护甲可附加 **符文结界**（维金），提供额外承伤层。

### 武器与词缀

| 阶段 | 常见词缀 |
|------|------|
| 剧情 | 面板高的锤/斧即可，附带物理点伤 |
| 过渡 | 技能等级（对应主技能）、攻击速度、晕眩/破甲相关 |
| 异界 | 生命、护甲%、抗性，暗金按 BD 选用 |

### 饰品

- 戒指/项链：**生命、四抗**；缺力量时补属性。
- 鞋：**30%+ 移动速度** 为近战职业常见目标词缀。
- 药剂：瞬间回复生命；部分 BD 用 **护甲/坚甲** 类药剂词条。

---

## 机制说明

1. 攻击命中可 **击退** 敌人；范围技能需注意走位。
2. 力量需求高的装备可用天赋 **博学** 或装备补属性。
3. 剧情装备奖励：武器面板、缺抗补抗。

## 通用开荒提示（国服）

1. 技能宝石：每升一级检查任务奖励与商人。
2. 装备词缀：生命、四抗、鞋 **30%+ 移动速度**、护甲。
3. 0.5 赛季：法罗任务；≤55 级护甲可附加 **符文结界**。
4. 属性不足：城镇 **兜帽人** 洗点。
5. 国服组队稀有掉落有独立系数；见版本备忘与 FAQ。
