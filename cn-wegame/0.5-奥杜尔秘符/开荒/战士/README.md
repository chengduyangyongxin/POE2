# 战士 · 0.5 开荒攻略索引

> 国服 · 奥杜尔秘符赛季 · 开荒向  
> 技能译名以 [POE2DB 中文](https://poe2db.tw/cn/) 为准

---

## 职业基础

| 项目 | 说明 |
|------|------|
| 主属性 | **力量** |
| 主武器 | **锤、斧、巨剑** 等近战双手/单手武器 |
| 防御倾向 | **护甲** + **生命**（全职业基础生命偏高） |
| 职业特性 | 攻击命中敌人时可将其 **击退** |
| 可选升华 | **泰坦**、**战争使者**、**奇塔弗匠师** |

战士招式力贯千钧，正面承伤能力强，适合喜欢近战、操作稳健的玩家。劣势是机动性一般，需注意走位与范围技能躲避。

---

## 技能体系

### 武器与技能类型

- 多数近战技能绑定 **力量武器**（锤/斧/剑），选 BD 前先确认技能要求的武器类型。
- 常见开荒方向：**猛击类**（泰坦）、**战吼 + 先祖**（战争使者）、**锻造/破甲**（奇塔弗匠师）。
- 部分技能可造成 **晕眩、破甲**，与战士高护甲形成「控住再打」的节奏。

### 精魂与光环

- 战士开荒精魂压力通常 **低于** 女巫/佣兵捷光环流，但战吼、先祖、部分增益仍占精魂。
- 前期优先保证 **主输出技能五连**，精魂够再开 **捷光环** 或战吼类常驻。

### 技能连接要点

| 概念 | 作用 |
|------|------|
| **猛击 / 重击** | 单体 BOSS 爆发，常配合晕眩链 |
| **战吼** | 范围削弱敌人或强化自身，战争使者核心 |
| **先祖之灵** | 额外伤害来源，可脱手输出 |
| **破甲** | 降低敌人护甲，团队与 solo 均有效 |

## 天赋与升华

### 天赋树特点

- 起点在力量侧，易点到 **生命、护甲、近战伤害、晕眩**。
- 开荒期：**生命 > 四抗 > 输出大点**；进异界再补护甲%、元素减伤。
- 力量够穿装备即可，多余点数可转生命或伤害。

### 三大升华方向

| 升华 | 定位 | 核心机制 |
|------|------|----------|
| **泰坦** | 坦克 + 猛击 | 晕眩、猛击强化、高护甲减伤、先祖神力 |
| **战争使者** | 战吼 + 范围爆发 | 先祖之灵、战吼使尸体爆炸、前线压制 |
| **奇塔弗匠师** | 锻造 + 破甲 | 自带锻造、武器品质、奇塔弗黑暗之力 |

**开荒建议**：追求容错选 **泰坦**；喜欢清图爆炸感选 **战争使者**；愿意投资装备制作选 **奇塔弗匠师**。

## 装备体系

### 防御思路

- **护甲** 是主防御层，胸甲、头盔优先高护甲基底 + 生命。
- 词条「**护甲值的 X% 也适用于元素伤害**」对战士价值极高。
- 0.5 赛季：≤55 级护甲积极做 **符文结界**（维金），相当于第二条血条。

### 武器与词缀

| 阶段 | 优先 |
|------|------|
| 剧情 | 面板高的锤/斧即可，附带物理点伤 |
| 过渡 | 技能等级（对应主技能）、攻击速度、晕眩/破甲相关 |
| 异界 | 生命、护甲%、抗性，暗金按 BD 选用 |

### 饰品

- 戒指/项链：**生命、四抗**；缺力量时补属性。
- 鞋：**30%+ 移动速度** 几乎刚需，近战职业走位压力大。
- 药剂：瞬间回复生命；部分 BD 用 **护甲/坚甲** 类药剂词条。

---

## 开荒注意事项

1. **击退** 有时会把怪打散，AOE 技能需注意站位。
2. 近战容易吃地面效果，药剂与生命池要留余量。
3. BOSS 战预留 **晕眩/控场** 窗口，勿贪刀。
4. 力量需求高的装备可用天赋 **博学** 或装备补属性。
5. 剧情任务装备奖励：优先 **武器面板**、缺抗补抗。

## 通用开荒提示（国服）

1. 技能宝石：每升一级检查任务奖励与商人，优先凑齐主技能孔位。
2. 装备：优先 **生命、四抗、移动速度（鞋 30%+）**、**护甲**。
3. 0.5 赛季：尽早做法罗任务，低等级护甲加 **符文结界**（55 级以下无惩罚）。
4. 属性不够：城镇 **兜帽人** 洗点。
5. 国服组队掉落削弱，独狼或固定队往往更划算。
