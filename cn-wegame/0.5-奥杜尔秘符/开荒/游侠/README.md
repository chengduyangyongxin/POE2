## 职业基础

| 项目 | 说明 |
|------|------|
| 主属性 | **敏捷** |
| 主武器 | **弓** |
| 防御倾向 | **闪避** + 生命 |
| 职业特性 | 箭矢 **飞行迅速且精准**；全职业 **远程攻击精准度** 最高 |

---

## 技能体系

### 武器与技能类型

- 弓系技能均依赖 **弓** 装备，换 BD 时注意武器基底（短弓/长弓等）。
- 核心玩法：**物理/元素箭**（锐眼）、**毒药/药剂联动**（追猎者）。
- 可混合 **火、冰、电** 属性箭矢，灵活应对抗性。

### 精魂与光环

- **捷光环**（如闪电之捷、冰霜之捷）在游侠后期很常见；精魂不足时主技能连接通常先于捷光环启用。
- 追猎者可将 **魔力药剂** 转化为毒药机制，与精魂占用需统筹。

### 技能连接要点

| 概念 | 作用 |
|------|------|
| **多重射击 / 分叉** | 清图范围与命中数 |
| **穿透 / 连锁** | 箭矢弹射，提升清图效率 |
| **陷阱 / 地雷** | 脱手伤害，部分 BD 核心 |
| **印记** | BOSS 单体增伤、回蓝 |

## 天赋与升华

### 天赋树特点

- 起点敏捷侧，易拿 **闪避、投射物伤害、攻速、移动速度**。
- 天赋侧常见投资：**闪避** 与 **生命** 并行。
- 智慧需求：部分辅助宝石需智慧，装备或天赋补少量即可。

### 两大升华方向

| 升华 | 玩法类型 | 核心机制 |
|------|------|----------|
| **锐眼** | 神射手 | 额外箭矢、高机动、投射物爆发 |
| **追猎者** | 药剂毒师 | 药剂转毒、叠毒、混沌/毒伤清图 |

---

## 装备体系

### 防御思路

- **闪避** 为主防御层；闪避未触发时由生命与抗性承伤。
- 生命池仍需堆叠，闪避非 100% 覆盖。
- 可选 **闪避 + 护甲 hybrid** 抬 0.5 **符文结界** 上限（若副修生存向）。

### 武器与词缀

| 阶段 | 常见词缀 |
|------|------|
| 剧情 | 弓的面板物理/元素点伤、攻速 |
| 过渡 | +投射物/弓技能等级、暴击（若 BD 需要） |
| 异界 | 生命、闪避、抗性；鞋类常见 **30%+ 移动速度** |

### 饰品

- 手套：攻速、点伤；项链：技能等级、生命。
- 追猎者额外关注：**药剂充能、中毒伤害、混沌抗**。

---

## 机制说明

1. 职业天赋提高箭矢 **精准度**；近身战需 **翻滚/扑击** 等位移技能。
2. 元素箭受怪物抗性影响；可配 **寒霜爆**、**元素要害** 等减抗技能。
3. 第二武器组（护身符 + 扑击）剧情中后期解锁，用于 BOSS 回蓝。
4. 剧情装备奖励：弓面板、缺抗补抗。

## 通用开荒提示（国服）

1. 技能宝石：每升一级检查任务奖励与商人。
2. 装备词缀：生命、四抗、鞋 **30%+ 移动速度**、闪避。
3. 0.5 赛季：法罗任务；≤55 级护甲可附加 **符文结界**。
4. 属性不足：城镇 **兜帽人** 洗点；智慧不足由装备补充。
5. 国服组队稀有掉落有独立系数；见版本备忘与 FAQ。
