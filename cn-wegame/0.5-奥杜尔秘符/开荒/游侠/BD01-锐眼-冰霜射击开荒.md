# BD01 · 锐眼 · 冰霜射击开荒

> 国服 0.5 奥杜尔秘符 · 剧情至异界初期  
> 难度：★★☆☆☆ · 造价：低

---

## 一、BD 概述

31 级后切 **冰霜射击** + **狙击**，配合 **冰霜之捷** 碎冰清图。锐眼 **无尽弹药、聚风** 是竞速向弓手开荒首选之一。

| 优势 | 劣势 |
|------|------|
| 冻结控场、碎冰清图爽 | 31 级前较弱 |
| 远程安全 | 身板脆，忌贴脸 |
| 箭矢精准度高 | 冰免疫怪需备用伤害 |

---

## 二、升华：锐眼

| 顺序 | 升华节点 | 作用 |
|------|----------|------|
| 1 | 投射物伤害小点 | 基础 |
| 2 | **无尽弹药** | 额外箭矢 |
| 3 | **聚风** / 近距增伤 | 爆发 |
| 4 | **偏转**（0.5） | 闪避生存 |
| 5+ | 暴击、攻速 | 异界 |

---

## 三、技能

### 3.1 学习顺序

| 阶段 | 技能 |
|------|------|
| 1–30 级 | **闪电箭矢** + **引雷针** + **雷霆之捷** |
| 31 级+ | **冰霜射击** + **狙击** |
| 全程 | **寒霜爆**、**冰冻印记** |
| 中后期 | **冰霜之捷** 替换雷霆之捷 |

### 3.2 技能连接

**过渡期**

```
闪电箭矢 — 分叉 I — 闪电协调 I
```

**主输出**

```
冰霜射击 — 弹幕 I — 冰尖箭 I — 审议
```

```
狙击 — 机会之窗 I — 效能 I（BOSS）
```

### 3.3 循环

- 清图：冰霜射击穿透 → 冰霜之捷碎冰。
- BOSS：寒霜爆 → 冰冻印记 → 狙击连射。
