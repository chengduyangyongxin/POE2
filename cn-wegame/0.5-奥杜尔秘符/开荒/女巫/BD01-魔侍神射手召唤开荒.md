# BD01 · 魔侍神射手 · 召唤开荒

> 国服 0.5 奥杜尔秘符 · 巫妖 / 驱炎使通用  
> 难度：★☆☆☆☆ · 造价：低 · 暗金依赖：几乎无

---

## 一、BD 概述

以 **魔侍神射手** 为核心，通过号令 **瓦斯之箭** 配合 **烈焰之墙** 引爆，实现高额清图与 BOSS 爆发。0.5 版本召唤 bug 修复后伤害稳定，社区实测 **4–5 小时通关剧情** 可期。

| 优势 | 劣势 |
|------|------|
| 站着走路，召唤物代打 | 精魂前期紧张 |
| 几乎不吃装备，+1 召唤等级即起飞 | 瓦斯之箭需配合点燃/火焰墙 |
| 巫妖、驱炎使均可，升华可换 | 屏幕怪多时可略卡 |

---

## 二、升华选择

### 方案 A：巫妖（推荐独狼、想后期继续召唤）

| 顺序 | 升华节点 | 作用 |
|------|----------|------|
| 1 | **通灵导体** | 魔力 >35% 时，你与友军（召唤物）伤害大幅提升 |
| 2 | 魔力/生命小点 ×2 | 支撑导体阈值 |
| 3 | **无魂形态** | 魔力不再自然回复，改为按 **生命 6%/秒** 回蓝，解决导体耗蓝 |
| 4 | **灵魂破裂** | 诅咒敌人死亡 33% 混沌爆炸，清图加速 |
| 5 | **结晶护命匣** | 珠宝护命匣，后期强 |
| 6+ | 古神赋力 / 力量的代价 | 异界再补 |

> 剧情经常缺蓝：第二次试炼可先点 **无魂形态**，再回头点满导体相关。

### 方案 B：驱炎使（推荐火焰联动、炽焰猎犬路线）

| 顺序 | 升华节点 | 作用 |
|------|----------|------|
| 1 | **忠诚的炽焰猎犬** | 获得 **召唤炽焰猎犬**，配合 **忠诚**、**肉盾 II**、**元素大军** |
| 2 | 生命小点 ×2 | 贝达特系节点前置 |
| 3 | **贝达特的凝视** 或 **贝达特之手** | 生命转魔力/护盾，撑精魂与生存 |
| 4 | **重塑身躯** | 元素伤转火焰，配合点燃 |
| 5+ | 修罗入魔 / 火卜契约 | 异界再选 |

**第一次升华**若走纯魔侍神射手、暂不猎犬：可先点驱炎使 **法术伤害** 小点，第二次再拿 **炽焰猎犬**（见 [BD03](./BD03-驱炎使炽焰猎犬开荒.md)）。

## 三、技能

### 3.1 剧情学习顺序

| 阶段 | 技能 | 说明 |
|------|------|------|
| 1 级 | **瘟疫** | 伦利处未切割宝石；清小怪 |
| 清费尔 | **烈焰之墙** | 拾取技能宝石并切割 |
| 6 级左右 | **魔侍神射手** | 主召唤；2 级宝石优先升级它 |
| 红谷 | **寒霜爆** | BOSS 曝露 |
| 铁锈王 | **精华吸取** | 过渡清图，弗蕾树洞后换 **愤怒狂灵** |
| 拉赫兰 | **扑击** | 第二武器组 + 护身符，印记回蓝 |
| 第三幕 | **元素要害** | BOSS 前施放 |
| 第三幕末 | 绑定 spectre（可选） | 乌扎尔 5 级区域，50 精魂，帮点燃瓦斯云 |

**升级优先级**：魔侍神射手 > 烈焰之墙 > 寒霜爆

### 3.2 技能连接

#### 主召唤 · 魔侍神射手

```
魔侍神射手 — 火焰协调 — 多重射击 I（→ II）
```

| 辅助 | 说明 |
|------|------|
| 火焰协调 | **核心**：瓦斯之箭引爆时产生火焰爆炸，清屏关键 |
| 多重射击 I/II | 每次号令射出更多瓦斯云，范围与伤害 |

精魂够用时加：

```
魔侍神射手 — 火焰协调 — 多重射击 II — 狂噬 I — 肉盾 II
```

- **狂噬 I**：召唤物 +30% 伤害（也更易死，配合肉盾）
- **肉盾 II**：召唤物承伤，你更肉

#### 清图过渡（弗蕾树洞前）

```
精华吸取 — 范围扩大 I — 效能 I
```

拿到 **愤怒狂灵** 后弃用精华吸取。

#### 精魂 · 愤怒狂灵（弗蕾树洞后）

```
愤怒狂灵 — 烈焰之墙 — 狂噬 I
```

- 召唤时踩 **烈焰之墙**，狂灵穿火增伤并点燃。

#### 辅助法术

```
寒霜爆 — 效能 I
元素要害 — 效能 I（第三幕后）
```

#### 位移 / 印记

```
扑击 — 虹吸印记 II
```

- 装备 **护身符** 于第二武器组；打 BOSS 周期性扑击回蓝。

#### 驱炎使可选精魂

```
召唤炽焰猎犬 — 忠诚 — 肉盾 II — 元素大军
```

### 3.3 输出循环

**清图（早期）**

1. **瘟疫** 点杀或传染小怪。
2. 6 级后：**魔侍神射手** 常驻，走路即可。

**清图（有狂灵后）**

1. 开 **愤怒狂灵** + **烈焰之墙**。
2. 魔侍神射手自动射击；精英怪补号令 **瓦斯之箭**。

**BOSS**

1. 施放 **寒霜爆**（元素曝露）。
2. 第三幕后先 **元素要害**。
3. 脚下铺 **烈焰之墙**。
4. 号令 **瓦斯之箭** 到 BOSS 脚下（点燃后引爆）。
5. 重复 3–4；冷却中 **扑击** 上印记。
6. BOSS 废话时可预铺烈焰之墙与瓦斯云。

## 四、天赋

### 4.1 加点思路

- **召唤物伤害、生命、范围、速度**
- **智慧、能量护盾、生命**
- 早期不必点太多诅咒/混沌，先保召唤集群
- 巫妖：中等 **魔力上限**，维持导体 >35%
- 驱炎使：可点 **生命** 支撑贝达特转化

### 4.2 优先集群

| 类别 | 方向 |
|------|------|
| 召唤 | 召唤物伤害、生命、攻击/施法速度、范围 |
| 防御 | 能量护盾、护盾回复、生命、混沌抗 |
| 精魂 | 精魂上限（愤怒狂灵、炽焰猎犬需要） |
| 诅咒 | 巫妖后期：诅咒效果、范围 |
| 属性 | 智慧；**博学** 降装备需求 |

### 4.3 等级阶段

| 阶段 | 等级 | 重点 |
|------|------|------|
| 第一幕 | 1–15 | 瘟疫、智慧、生命 |
| 第二幕 | 16–35 | 召唤集群、四抗、精魂 |
| 第三幕 | 36–50 | 召唤伤害、诅咒（巫妖） |
| 异界 | 60+ | 补防御、+1 召唤装备 |

## 五、装备

### 5.1 武器

| 阶段 | 要求 |
|------|------|
| 第一幕 | 商人 **咔哒权杖** 等，自带 **魔侍武士**（不占精魂的额外召唤） |
| 全程 | **+1 全部召唤技能等级**（权杖/短杖）> 法术点伤 |
| 过渡 | 法杖：召唤等级、施法速度、智慧 |

### 5.2 防具

| 部位 | 优先级 |
|------|--------|
| 头盔 | **+1 召唤技能等级** > 生命/护盾/抗性 |
| 胸甲 | 能量护盾 + 生命 hybrid |
| 手套/鞋 | 生命、抗性；鞋 **移速 30%+** |
| 项链 | **+1 召唤技能等级**、生命、抗性 |

### 5.3 饰品与药剂

- 戒指：生命、抗性、智慧；后期可加召唤伤害
- 药剂：瞬间回复生命；魔力药剂高回复
- 护身符：任意（仅用于 **扑击** 第二武器组）

### 5.4 商店搜索技巧（国服）

可用过滤器/市集搜：`法术伤害`、`召唤`、`最大精魂`、`护盾` 等组合。

### 5.5 0.5 赛季机制

- ≤55 级护甲打 **符文结界**，召唤流站后排也受益。
- 遗物优先召唤相关词缀或防御。

## 七、异界过渡

1. 精魂够：加 **绑定幽魂** 或继续扩召唤（魔侍风暴法师等）。
2. 巫妖：完善诅咒链、**结晶护命匣** 珠宝。
3. 驱炎使：可转向 [BD03 炽焰猎犬](./BD03-驱炎使炽焰猎犬开荒.md) 深化。
4. 追求一键清图：评估是否转 [BD02 瘟疫尖刺](./BD02-瘟疫尖刺流开荒.md) 或继续堆 +1 召唤等级。
