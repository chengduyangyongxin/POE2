# BD02 · 瘟疫尖刺流 · 开荒

> 国服 0.5 奥杜尔秘符 · 推荐升华：**巫妖**（后期可转 **命源法师**）  
> 难度：★★★☆☆ · 造价：低中 · **双武器组**

---

## 一、BD 概述

**瘟疫尖刺**（社区俗称「瘟疫骨刺」「灵吸骨刺」）用 **瘟疫 + 精华吸取** 清小怪，用 **尖刺风暴** 叠 **穿刺** 打 BOSS。通过 **活体闪电** 等攻击触发拔刺，单体爆发极高，剧情 **6–7 小时进异界** 的速通向 BD。

| 优势 | 劣势 |
|------|------|
| 清图快、传染机制爽 | 需两套武器 + 天赋页 |
| 低造价通关剧情 | 穿刺机制需理解 |
| 后期可无缝转 **命源法师** COC/血法 | 精魂与魔力管理 |

---

## 二、升华：巫妖

剧情阶段以巫妖为主（更肉、诅咒联动）；进异界后可洗 **命源法师** 走血法。

### 开荒升华顺序（巫妖）

| 顺序 | 节点 | 作用 |
|------|------|------|
| 1 | **通灵导体** | 高蓝线伤害 |
| 2 | **无魂形态** | 回蓝 |
| 3 | **灵魂破裂** | 清图爆炸 |
| 4 | 能量护盾/生命小点 | 生存 |
| 5 | **结晶护命匣** | 珠宝位 |

### 后期转型：命源法师（异界，可选）

| 节点 | 作用 |
|------|------|
| **皮开肉绽** | 法术基础暴击率固定 **15%** |
| **活力虹吸** | 法术伤害 20% 转生命偷取 |
| **滴血为咒** | 生命残片机制 |
| **弥久伤痕** | 减伤缓冲 |

> 剧情不必强行命源法师；**驱炎使** 走火卜契约也可过渡，但尖刺流主流是巫妖→血法。

## 三、技能

### 3.1 双武器组分工

| 武器组 | 绑定天赋 | 主技能 | 伤害类型 |
|--------|----------|--------|----------|
| **套装 1** | 第一套法杖 | 瘟疫、精华吸取、诅咒光环 | **混沌** 清图 |
| **套装 2** | 第二套法杖 | 尖刺风暴、暴击时施法链 | **物理** BOSS |

### 3.2 技能学习顺序

1. **瘟疫**（1 级）
2. **精华吸取**（早期清图）
3. **尖刺风暴**（约 14 级，5 级未切割技能切出）
4. 第一幕末：法杖镶 **低阶暴风符文** 或找 **额外闪电伤害** 前缀（提早拔刺）
5. 58 级左右：**暴击时施法** + **风暴漩涡** / **天雷之珠** / **活体闪电 II**（套装 2）

### 3.3 技能连接

#### 清图 · 套装 1

**瘟疫**

```
瘟疫 — 释放 — 增幅光环 I
```

或前期：

```
瘟疫 — 范围扩大 I — 效能 I
```

**精华吸取**（尖刺风暴拿到前）

```
精华吸取 — 范围扩大 I — 效能 I — 混沌专精（五连后）
```

**诅咒（精魂够用时）**

```
元素要害 — 释放 — 增幅光环 I
```

或巫妖后期：**绝望** / **易燃** 等（按精魂调整）。

#### 攻坚 · 套装 2 · 尖刺风暴

**前期（第一章后，简易拔刺）**

```
尖刺风暴 — 野性残暴 I — 极盛 I — 活体闪电 I
```

- 法杖需 **闪电 gain 伤** 或 **暴风符文**，让尖刺带闪电伤 → 产 **活体闪电** 拔刺。

**中期（四连+）**

```
尖刺风暴 — 野性残暴 I — 极盛 I — 审议 — 从容不迫
```

**后期（58 级+，暴击时施法）**

```
尖刺风暴 — 野性残暴 II — 极盛 II — 细思施法 — 审议
```

独立触发链（仅套装 2 开启）：

```
暴击时施法 — 风暴漩涡 — 天雷之珠 — 活体闪电 II
```

- 尖刺风暴高暴击 → 不断召唤活体闪电 → **攻击** 触发 **穿刺** 引爆。

#### 辅助

```
寒霜爆 — 效能 I
扑击 — 虹吸印记 II（护身符第二套）
```

### 3.4 穿刺 / 拔刺原理（必读）

1. **尖刺风暴** 引导蓄刺，松手射出；命中叠 **穿刺**（本技能 **200% 穿刺幅度**）。
2. 穿刺需后续 **攻击命中** 才能引出额外物理伤。
3. **活体闪电** 的攻击可稳定拔刺；故套装 2 法杖要带闪电伤害来源。
4. BOSS 战：先引导满刺 → 松手 → 活体闪电/暴击时施法连段引爆。

### 3.5 输出循环

**清图**

1. **瘟疫** 传染（或精华吸取点杀）。
2. 小怪不用尖刺，省蓝。

**BOSS**

1. **寒霜爆** + **元素要害**（有则放）。
2. 切换到 **套装 2**。
3. 引导 **尖刺风暴** 满 20 刺（或够层数）。
4. 松手 → 若已链 **暴击时施法**，自动产活体闪电拔刺。
5. 早期无 COC：手动在尖刺上链 **活体闪电 I** 辅助。

## 四、天赋

### 4.1 套装 1（混沌清图）

- 混沌伤害、持续伤害、法术伤害
- 诅咒效果（巫妖）
- 智慧、魔力、精魂

### 4.2 套装 2（物理 BOSS）

- 物理法术、穿刺相关、暴击
- **细思施法**、施法速度
- 暴击时施法集群

### 4.3 通用防御

- 能量护盾 + 生命
- 混沌抗性（剧情建议 ≥10%）
- 元素转护盾（驱炎使贝达特线可选）

### 4.4 原则

- **全局混沌增伤** 放套装 1 常规树。
- **仅尖刺/暴击施法受益** 的大点放套装 2 武器页。

## 五、装备

### 5.1 武器

| 套装 | 词缀优先 |
|------|----------|
| 套装 1（混沌） | +法术/混沌技能等级、施法速度、混沌法术伤害 |

- 血药瞬间回复
- 蓝药高回复；BOSS 战注意魔力
