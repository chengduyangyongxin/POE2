## 职业基础

| 项目 | 说明 |
|------|------|
| 主属性 | **智慧** |
| 主武器 | **法杖、短杖、权杖** |
| 防御倾向 | **能量护盾** + 生命 hybrid |
| 职业特性 | 对 **混沌伤害** 具备天然抗性 |
| 可选升华 | **巫妖**、**驱炎使**、**命源法师** |

法术与召唤双线并存；常见玩法含召唤流与瘟疫尖刺流。召唤玩法前期 **精魂** 占用较高；瘟疫尖刺使用 **双武器组** 与穿刺机制。

## 技能体系

### 武器与技能类型

- **法术**：瘟疫、精华吸取、尖刺风暴等，绑定法杖，吃 **施法速度、法术/混沌等级**。
- **召唤**：魔侍神射手、愤怒狂灵、炽焰猎犬等，常绑 **权杖/短杖**，吃 **+召唤技能等级**。
- 同一角色可混玩法术 + 召唤；**精魂** 为硬约束，通常单线构筑主技能。

### 精魂与光环

- **精魂** = 可常驻技能/召唤的上限；召唤物、愤怒狂灵、捷光环都占精魂。
- 精魂分配顺序（常见）：**主召唤或主法术** → 关键辅助 → 诅咒/第二召唤 → 捷光环。
- **释放 + 增幅光环** 可把诅咒/瘟疫转为光环，省操作但占精魂。

### 技能连接要点

| 概念 | 作用 |
|------|------|
| **号令**（魔侍神射手） | 控制召唤物释放技能（如瓦斯之箭） |
| **烈焰之墙** | 火焰地面，联动点燃与狂灵 |
| **瘟疫传染** | 混沌 DOT 清小怪，几乎不占操作 |
| **穿刺/拔刺**（尖刺风暴） | 物理 BOSS 机制，需活体闪电等触发 |
| **诅咒**（元素要害等） | BOSS 减抗，巫妖可深化诅咒链 |

### 双武器组

- 剧情中后期解锁 **第二武器组**：常见 **护身符 + 扑击 + 虹吸印记**，BOSS 回蓝。
- **BD02 瘟疫尖刺** 必须双武器组：混沌法杖清图 + 物理法杖 BOSS。

## 天赋与升华

### 天赋树特点

- 智慧起点，易点 **能量护盾、生命、召唤伤害、混沌/诅咒**。
- 女巫天赋含 **混沌抗**；剧情阶段可堆叠至约 **10%+** 以应对混沌伤害。
- 召唤流常见词缀：**召唤伤害、生命、范围、精魂上限**。

### 三大升华方向

| 升华 | 玩法类型 | 核心机制 |
|------|------|----------|
| **巫妖** | 召唤 + 诅咒 | 通灵导体、无魂形态、灵魂破裂、珠宝护命匣 |
| **驱炎使** | 火焰 + 召唤 | 炽焰猎犬、热量积蓄、火转伤、地狱烈焰体系 |
| **命源法师** | 血法 | 生命能量、命运精华；BD02 可转入 |

## 装备体系

### 防御思路

- **能量护盾 + 生命** hybrid；纯 ES 构筑对一击伤害敏感。
- 召唤流站位偏后，仍依赖 **四抗** 与生命池。
- 0.5：≤55 级护甲可附加 **符文结界**。

## 机制说明

1. **+1 召唤技能等级** 词缀对召唤流伤害影响显著（权杖、头盔、项链）。
2. 精魂不足时，可先停用次要召唤，保留 **魔侍神射手** 主连接。
3. 瓦斯之箭依赖 **点燃** 与 **烈焰之墙**；需连接 **火焰协调** 等辅助。
4. 巫妖 **通灵导体** 消耗魔力；可点 **无魂形态** 或使用 **扑击** 回蓝。
5. 瘟疫尖刺的 **穿刺** 机制见 BD02「拔刺原理」章节。

## 通用开荒提示（国服）

1. 技能宝石：每升一级检查任务奖励与商人；剧情装备奖励见各 BD 第六节。
2. 召唤流装备词缀：**+1 召唤技能等级**（权杖、头盔、项链）。
3. 防御词缀：生命、四抗、鞋 **30%+ 移动速度**、能量护盾 hybrid。
4. 精魂：主召唤连接通常先于捷光环启用。
5. 0.5 赛季：法罗任务；≤55 级护甲可附加 **符文结界**。
6. 混沌抗：女巫有基础混沌抗，剧情可堆至 **10%+**。
