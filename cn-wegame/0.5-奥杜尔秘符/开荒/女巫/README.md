## 通用开荒提示（国服）

1. **召唤物等级**：权杖/短杖、头盔、项链优先找 **+1 召唤技能等级**，收益极高。
2. 每级检查商人：**未切割技能/辅助宝石**、带词缀的法杖。
3. 属性不够：城镇 **兜帽人** 洗点；智慧不足优先装备补。
4. 精魂：召唤流前期紧，先保主召唤，捷光环后期再加。
5. 0.5 赛季：法罗任务、低等级护甲 **符文结界**（≤55 级无惩罚）。
6. 混沌抗：女巫有基础混沌抗，剧情仍建议凑到 **至少 10%+** 再硬扛混沌怪。
