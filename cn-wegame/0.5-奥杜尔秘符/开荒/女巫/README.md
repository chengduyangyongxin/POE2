# 女巫 · 0.5 开荒攻略索引

> 国服 · 奥杜尔秘符赛季 · 开荒向  
> 技能译名以 [POE2DB 中文](https://poe2db.tw/cn/) 为准

---

## 职业基础

| 项目 | 说明 |
|------|------|
| 主属性 | **智慧** |
| 主武器 | **法杖、短杖、权杖** |
| 防御倾向 | **能量护盾** + 生命 hybrid |
| 职业特性 | 对 **混沌伤害** 具备天然抗性 |
| 可选升华 | **巫妖**、**驱炎使**、**命源法师** |

女巫是 0.5 国服社区公认的 **T0 开荒职业** 之一：召唤流容错高、对装备要求低；瘟疫尖刺流清图快。劣势是召唤前期 **精魂紧张**、瘟疫尖刺需管理 **双武器组** 与穿刺机制。

## 技能体系

### 武器与技能类型

- **法术**：瘟疫、精华吸取、尖刺风暴等，绑定法杖，吃 **施法速度、法术/混沌等级**。
- **召唤**：魔侍神射手、愤怒狂灵、炽焰猎犬等，常绑 **权杖/短杖**，吃 **+召唤技能等级**。
- 同一角色可混玩法术 + 召唤，但 **精魂** 是硬约束，开荒建议主修一条线。

### 精魂与光环

- **精魂** = 可常驻技能/召唤的上限；召唤物、愤怒狂灵、捷光环都占精魂。
- 开荒优先级：**主召唤或主法术** → 关键辅助 → 诅咒/第二召唤 → 捷光环。
- **释放 + 增幅光环** 可把诅咒/瘟疫转为光环，省操作但占精魂。

### 技能连接要点

| 概念 | 作用 |
|------|------|
| **号令**（魔侍神射手） | 控制召唤物释放技能（如瓦斯之箭） |
| **烈焰之墙** | 火焰地面，联动点燃与狂灵 |
| **瘟疫传染** | 混沌 DOT 清小怪，几乎不占操作 |
| **穿刺/拔刺**（尖刺风暴） | 物理 BOSS 机制，需活体闪电等触发 |
| **诅咒**（元素要害等） | BOSS 减抗，巫妖可深化诅咒链 |

### 双武器组

- 剧情中后期解锁 **第二武器组**：常见 **护身符 + 扑击 + 虹吸印记**，BOSS 回蓝。
- **BD02 瘟疫尖刺** 必须双武器组：混沌法杖清图 + 物理法杖 BOSS。

## 天赋与升华

### 天赋树特点

- 智慧起点，易点 **能量护盾、生命、召唤伤害、混沌/诅咒**。
- 女巫 **混沌抗** 天赋减轻配装，剧情仍建议 **10%+ 混沌抗**。
- 召唤流：**召唤伤害、生命、范围、精魂上限** 优先。

### 三大升华方向

| 升华 | 定位 | 核心机制 |
|------|------|----------|
| **巫妖** | 召唤 + 诅咒 | 通灵导体、无魂形态、灵魂破裂、珠宝护命匣 |
| **驱炎使** | 火焰 + 召唤 | 炽焰猎犬、热量积蓄、火转伤、地狱烈焰体系 |
| **命源法师** | 血法 | 生命能量、命运精华、后期向（常由 BD02 转入） |

**开荒建议**：不确定选 **BD01 巫妖/驱炎使召唤**；速通剧情 **BD02 巫妖瘟疫尖刺**。

## 装备体系

### 防御思路

- **能量护盾 + 生命** hybrid，勿纯 ES 裸奔剧情。
- 召唤流站后排，仍要堆 **四抗** 与生命池。
- 0.5：**符文结界** 对低等级护甲友好，≤55 级积极使用。

### 武器与词缀

| 流派 | 武器优先 |
|------|----------|
| 召唤 | **+1 全部召唤技能等级** > 法术点伤；权杖带 **魔侍武士** 更佳 |
| 瘟疫尖刺 | 混沌套：法术/混沌等级；物理套：**闪电 gain 伤** / **暴风符文** |
| 驱炎使 | 同召唤 + **火焰伤害、生命** |

### 防具与饰品

| 部位 | 召唤流优先 |
|------|------------|
| 头盔/项链 | **+1 召唤技能等级** |
| 胸甲 | ES + 生命 |
| 鞋 | **30%+ 移速**、生命、抗性 |
| 戒指 | 生命、抗性、智慧 |

### 剧情任务奖励

见各 BD「**六、剧情任务奖励（建议）**」：技能宝石、武器、抗性选择。

## 开荒注意事项

1. **+1 召唤等级** 收益极高，权杖/头/链优先搜这个词缀。
2. 精魂不够时先砍次要召唤，保 **魔侍神射手** 主连接。
3. 瓦斯之箭需 **点燃 + 烈焰之墙** 联动，缺火焰协调辅助伤害会断档。
4. 巫妖 **通灵导体** 耗蓝，记得点 **无魂形态** 或带扑击回蓝。
5. 瘟疫尖刺 **穿刺** 机制需阅读 BD02「拔刺原理」，否则 BOSS 伤害偏低。

## 通用开荒提示（国服）

1. 技能宝石：每升一级检查任务奖励与商人；**剧情装备奖励**见各 BD 第六节。
2. **召唤物等级**：权杖/头/链优先 **+1 召唤技能等级**。
3. 装备：优先 **生命、四抗、移动速度（鞋 30%+）**、能量护盾 hybrid。
4. 精魂：召唤流前期紧，先保主召唤，捷光环后期再加。
5. 0.5 赛季：尽早做法罗任务，低等级护甲加 **符文结界**（55 级以下无惩罚）。
6. 混沌抗：女巫有基础混沌抗，剧情仍建议 **至少 10%+**。
