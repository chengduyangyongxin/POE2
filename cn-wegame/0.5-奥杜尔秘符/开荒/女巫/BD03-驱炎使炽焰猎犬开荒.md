# BD03 · 驱炎使 · 炽焰猎犬召唤

> 国服 0.5 奥杜尔秘符 · 升华：**驱炎使**  
> 难度：★★☆☆☆ · 造价：低 · 火焰 / 地狱烈焰体系

## 一、BD 概述

在 [BD01 魔侍神射手](./BD01-魔侍神射手召唤开荒.md) 基础上，以 **驱炎使** 升华为核心，强化 **召唤炽焰猎犬**、**地狱烈焰** 与点燃联动。适合喜欢火焰视觉、愿意管理 **走火入魔** / 地狱烈焰资源的玩家。

| 对比 BD01 | 说明 |
|-----------|------|
| 伤害类型 | 更偏 **火焰/点燃** |
| 生存 | 贝达特系 **生命转魔力/护盾** |
| 操作 | 需关注地狱烈焰层数（中后期） |
| 清图 | 魔侍神射手 + 猎犬双召唤，范围足 |

## 二、升华：驱炎使

### 推荐加点顺序

| 顺序 | 节点 | 作用 |
|------|------|------|
| 1 | **忠诚的炽焰猎犬** | 精魂技能 **召唤炽焰猎犬** |
| 2 | 生命小点 ×3 | 前置贝达特节点 |
| 3 | **贝达特的凝视** | 保留 25% 生命 → 魔力上限 |
| 4 | **贝达特之手** | 保留生命 → **能量护盾** |
| 5 | **重塑身躯** | 元素伤部分转火焰，配合点燃 |
| 6 | **修罗入魔** | 获得 **走火入魔** 技能，变身增伤 |
| 7+ | **火卜契约** / **沸腾之躯** | 异界：地狱烈焰体系 |

> 第一次升华若精魂不够同时猎犬+狂灵：可先点 **法术伤害** 小点，第二次再拿 **炽焰猎犬**。

### 核心机制

- **炽焰猎犬**：点燃周围敌人，配合魔侍神射手 **火焰协调 + 瓦斯之箭**。
- **贝达特系**：用生命换魔力/护盾/精魂，需堆 **生命上限**。
- **地狱烈焰**（火卜契约后）：技能消耗地狱烈焰而非魔力，需习惯层数管理。

## 三、技能

### 3.1 在 BD01 基础上的调整

保留 BD01 主体：

- **魔侍神射手** — 火焰协调 — 多重射击 II — 狂噬 I — 肉盾 II
- **愤怒狂灵** — 烈焰之墙 — 狂噬 I
- **寒霜爆**、**元素要害**、**扑击**

### 3.2 驱炎使专属精魂

```
召唤炽焰猎犬 — 忠诚 — 肉盾 II — 元素大军
```

| 辅助 | 作用 |
|------|------|
| 忠诚 | 猎犬承伤顺序优化 |
| 肉盾 II | 召唤物更肉 |
| 元素大军 | 元素召唤物伤害 |

### 3.3 中后期可选

```
走火入魔 — 效能 I
```

- 地狱烈焰高时开 **走火入魔** 变身，配合 **沸腾之躯** 元素执政官（异界）。

### 3.4 输出循环

与 BD01 相同，额外：

1. 常驻 **炽焰猎犬**（精魂允许时）。
2. 进战先 **烈焰之墙**，猎犬 + 狂灵 + 神射手点燃联动。
3. BOSS：**寒霜爆** → **元素要害** → 烈焰之墙 → 瓦斯之箭。
4. 地狱烈焰体系成型后：用 **走火入魔** 打精英/BOSS 爆发窗口。
