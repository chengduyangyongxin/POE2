## 职业基础

| 项目 | 说明 |
|------|------|
| 主属性 | **智慧** |
| 主武器 | **法杖、短杖** |
| 防御倾向 | **能量护盾** + 生命 |
| 职业特性 | 对 **元素攻击** 具备天然抗性 |
| 可选升华 | **风暴编织者**、**塑时术师**、**瓦拉煞的门徒** |

魔巫是经典元素施法者，远程编织冰火雷法术。与女巫（混沌/召唤）相比更偏 **元素爆发** 与 **法术连锁**。

---

## 技能体系

### 武器与技能类型

- 法术技能绑定 **法杖/短杖**，关注 **施法速度、法术伤害、技能等级**。
- 常见方向：**闪电链/风暴**（风暴编织者）、**时空控场**（塑时术师）、**巨灵召唤**（瓦拉煞的门徒）。
- 元素法术需注意怪物 **元素抗性**，配减抗或多元素轮换。

### 精魂与光环

- 元素 **捷光环**、持续增益可能占大量精魂。
- 开荒顺序：主技能四/五连 → 防御光环 → 捷光环。
- 塑时术师部分机制与 **冷却** 相关，精魂与魔力双资源管理。

### 技能连接要点

| 概念 | 作用 |
|------|------|
| **连锁 / 穿透** | 清图核心（闪电等） |
| **冰缓 / 感电** | 控场与增伤，风暴编织者重点 |
| **法术暴击** | 部分 BD 依赖暴击时施法 |
| **召唤巨灵**（瓦拉煞） | 脱手元素伤害 + 护盾转护甲 |
| **时间冻结/减速**（塑时） | 控场与安全输出窗口 |

## 天赋与升华

### 天赋树特点

- 智慧起点，易点 **能量护盾、法术伤害、施法速度、元素**。
- 魔巫元素抗天赋可减轻配装压力，但仍需凑 **混沌抗**。
- 瓦拉煞门徒可走 **护盾转护甲**，天赋与装备方向与纯 ES 不同。

### 三大升华方向

| 升华 | 定位 | 核心机制 |
|------|------|----------|
| **风暴编织者** | 元素风暴 | 冰缓感电、元素风暴、快速施法 |
| **塑时术师** | 时空法师 | 冷却重置、减速、时间冻结 |
| **瓦拉煞的门徒** | 巨灵 + 护甲 ES | 元素巨灵、护盾变护甲 |

**开荒建议**：传统元素清图 **风暴编织者**；控场体验 **塑时术师**；想召唤+坦度 **瓦拉煞的门徒**。

---

## 装备体系

### 防御思路

- **能量护盾** 为主，配合 **护盾回复、魔力**（部分 BD 用护盾充能）。
- 瓦拉煞门徒可堆 **护甲** 作为第二防御层。
- 生命 hybrid 仍是开荒安全垫，纯 ES 玻璃需注意一击线。

### 武器与词缀

| 阶段 | 优先 |
|------|------|
| 剧情 | 法术点伤、施法速度、智慧 |
| 过渡 | +元素/法术技能等级、魔力上限 |
| 异界 | ES、生命、四抗（含混沌） |

### 饰品

- 戒指/项链：**生命、抗性、施法速度**；后期 **暴击/元素伤**。
- 药剂：魔力高回复；部分 BD 用 **魔力药剂与技能联动**（追猎者式玩法在魔巫较少）。

---

## 开荒注意事项

1. **元素抗** 是职业天赋，**混沌抗** 仍需手动堆。
2. 施法职业魔力消耗大，注意 **回蓝** 与 **扑击+印记**（第二武器组）。
3. 塑时术师对机制理解要求高，新手慎选。
4. 风暴类技能特效多，低配机器注意帧率。
5. 剧情装备奖励：法杖 **法术点伤/等级**，缺抗补抗。

## 通用开荒提示（国服）

1. 技能宝石：每升一级检查任务奖励与商人，优先凑齐主技能孔位。
2. 装备：优先 **生命、四抗、能量护盾**；鞋 **30%+ 移速**。
3. 0.5 赛季：尽早做法罗任务，低等级护甲加 **符文结界**。
4. 属性不够：城镇 **兜帽人** 洗点；智慧不足装备补。
5. 国服组队掉落削弱，独狼或固定队往往更划算。
