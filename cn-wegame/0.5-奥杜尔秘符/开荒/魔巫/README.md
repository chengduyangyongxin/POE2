## 职业基础

| 项目 | 说明 |
|------|------|
| 主属性 | **智慧** |
| 主武器 | **法杖、短杖** |
| 防御倾向 | **能量护盾** + 生命 |
| 职业特性 | 对 **元素攻击** 具备天然抗性 |
| 可选升华 | **风暴编织者**、**塑时术师**、**瓦拉煞的门徒** |

元素法术职业；主武器 **法杖/短杖**；天赋含 **元素抗性**。玩法分支含风暴、塑时、瓦拉煞巨灵等。

---

## 技能体系

### 武器与技能类型

- 法术技能绑定 **法杖/短杖**，关注 **施法速度、法术伤害、技能等级**。
- 常见方向：**闪电链/风暴**（风暴编织者）、**时空控场**（塑时术师）、**巨灵召唤**（瓦拉煞的门徒）。
- 元素法术需注意怪物 **元素抗性**，配减抗或多元素轮换。

### 精魂与光环

- 元素 **捷光环**、持续增益可能占大量精魂。
- 开荒顺序：主技能四/五连 → 防御光环 → 捷光环。
- 塑时术师部分机制与 **冷却** 相关，精魂与魔力双资源管理。

### 技能连接要点

| 概念 | 作用 |
|------|------|
| **连锁 / 穿透** | 清图核心（闪电等） |
| **冰缓 / 感电** | 控场与增伤，风暴编织者重点 |
| **法术暴击** | 部分 BD 依赖暴击时施法 |
| **召唤巨灵**（瓦拉煞） | 脱手元素伤害 + 护盾转护甲 |
| **时间冻结/减速**（塑时） | 控场与安全输出窗口 |

## 天赋与升华

### 天赋树特点

- 智慧起点，易点 **能量护盾、法术伤害、施法速度、元素**。
- 魔巫元素抗天赋可减轻配装压力，但仍需凑 **混沌抗**。
- 瓦拉煞门徒可走 **护盾转护甲**，天赋与装备方向与纯 ES 不同。

### 三大升华方向

| 升华 | 玩法类型 | 核心机制 |
|------|------|----------|
| **风暴编织者** | 元素风暴 | 冰缓感电、元素风暴、快速施法 |
| **塑时术师** | 时空法师 | 冷却重置、减速、时间冻结 |
| **瓦拉煞的门徒** | 巨灵 + 护甲 ES | 元素巨灵、护盾变护甲 |

---

## 装备体系

### 防御思路

- **能量护盾** 为主，配合 **护盾回复、魔力**（部分 BD 用护盾充能）。
- 瓦拉煞门徒可堆 **护甲** 作为第二防御层。
- 生命 hybrid 与能量护盾并用；纯 ES 构筑对一击伤害敏感。

### 武器与词缀

| 阶段 | 常见词缀 |
|------|------|
| 剧情 | 法术点伤、施法速度、智慧 |
| 过渡 | +元素/法术技能等级、魔力上限 |
| 异界 | ES、生命、四抗（含混沌） |

### 饰品

- 戒指/项链：**生命、抗性、施法速度**；后期 **暴击/元素伤**。
- 药剂：魔力高回复；部分 BD 用 **魔力药剂与技能联动**（追猎者式玩法在魔巫较少）。

---

## 机制说明

1. 职业天赋提供 **元素抗**；**混沌抗** 需由装备与天赋补充。
2. 施法消耗魔力；可配 **扑击+印记**（第二武器组）回蓝。
3. 塑时术师涉及冷却与时间机制，见对应 BD 文档。
4. 剧情装备奖励：法杖法术点伤/等级、缺抗补抗。

## 通用开荒提示（国服）

1. 技能宝石：每升一级检查任务奖励与商人。
2. 装备词缀：生命、四抗、能量护盾、鞋 **30%+ 移动速度**。
3. 0.5 赛季：法罗任务；≤55 级护甲可附加 **符文结界**。
4. 属性不足：城镇 **兜帽人** 洗点；智慧由装备补充。
5. 国服组队稀有掉落有独立系数；见版本备忘与 FAQ。
