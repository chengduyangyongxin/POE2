# BD01 · 瓦拉煞门徒 · 纠缠植物开荒

> 国服 0.5 奥杜尔秘符 · 剧情至异界初期  
> 难度：★★☆☆☆ · 造价：低 · **魔巫 T0 开荒**

---

## 一、BD 概述

**纠缠** + **鞭笞藤蔓** + **雷霆风暴**「浇水」清图，瓦拉煞门徒 **权柄工具** 支持法杖+法器双持。一键浇花，操作量极低。

| 优势 | 劣势 |
|------|------|
| 脱手清图、省心 | 混沌抗要堆 |
| 低造价 | BOSS 需铺藤蔓 |
| 与德鲁伊植物流技能通用 | 精魂够才上暴击时施放 |

## 三、技能

### 3.1 学习顺序

1. **瘟疫** + **精华吸取**（1–12 级过渡）
2. **纠缠**（核心）
3. **鞭笞藤蔓**
4. **雷霆风暴**（浇水）
5. 精魂够：**暴击时施法** 链

### 3.2 技能连接

```
纠缠 — 加速生长 I — 分叉裂隙 I — 审议
```

```
雷霆风暴 — 范围扩大 I（对植物区域施放）
```

后期：

```
暴击时施法 — 雷霆风暴 — 活体闪电 II
```

### 3.3 循环

- 纠缠 → 藤蔓生长 → 雷霆风暴浇满屏。
- BOSS：纠缠 → 藤蔓 → 风暴 → 补法术。
