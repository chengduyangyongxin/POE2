## 一、BD 概述

**闪电战矛** 一键清图，BOSS 用 **雷霆风暴** + **灰烬连射** 打感电爆发。亚马逊战技强化，操作比旋龙简单。

| 优势 | 劣势 |
|------|------|
| 技能直观、清图快 | 单体机制不如旋龙精细 |
| 格挡 + 闪避 | 精魂占用中等 |
| BOSS 爆发高 | 闪电免疫需备用 |
