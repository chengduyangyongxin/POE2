## 一、BD 概述

**爆裂战矛** + **闪电战矛** + **雷鸣跃击**，仪祭师 **戒指 flat 伤、护符 uptime** 放大爆发。适合熟悉战矛、愿意投资装备的玩家。

| 优势 | 劣势 |
|------|------|
| 爆发极高 | 造价与操作要求高 |
| 仪祭师后期强 | 开荒期不如旋龙稳 |
| 机动性好 | 需管理护符与戒指词缀 |
