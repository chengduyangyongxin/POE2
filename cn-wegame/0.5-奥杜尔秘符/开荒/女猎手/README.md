# 女猎手 · 0.5 开荒攻略索引

> 国服 · 奥杜尔秘符赛季 · 开荒向  
> 技能译名以 [POE2DB 中文](https://poe2db.tw/cn/) 为准

---

## 职业基础

| 项目 | 说明 |
|------|------|
| 主属性 | **敏捷** |
| 主武器 | **战矛** |
| 防御倾向 | **闪避** + **格挡** + 生命 |
| 职业特性 | **训练有素**，善于 **格挡** 袭来的攻击 |
| 可选升华 | **仪祭师**、**亚马逊**、**灵魂行者** |

战矛近战/投掷混合；职业天赋提供 **格挡**。0.5 版本含 **灵魂行者** 升华（魂灵与伙伴机制）。

---

## 技能体系

### 武器与技能类型

- 战矛技能涵盖 **戳刺、横扫、投掷** 等多种形态，需按 BD 选主技能。
- 可 **近身搏杀** 也可 **中距离投矛**；战矛技能偏近战敏捷路线。
- **格挡** 与部分技能联动，是生存核心之一。

### 精魂与光环

- 仪祭师、灵魂行者可能占用较多精魂（仪式、魂灵、伙伴）。
- 精魂不足时主输出技能通常先于仪式/召唤类精魂技能启用。

### 技能连接要点

| 概念 | 作用 |
|------|------|
| **战矛戳刺/横扫** | 近战主输出 |
| **投掷** | 中远程清图、风筝 |
| **格挡回复 / 格挡率** | 生存与伤害联动 |
| **仪式/献祭**（仪祭师） | 以代价换爆发 |
| **魂灵/伙伴**（灵魂行者） | 额外伤害与辅助 |

---

## 天赋与升华

### 天赋树特点

- 敏捷起点，可点到 **格挡、闪避、长柄/矛伤害、生命**。
- 女猎手天赋含 **格挡** 集群。
- 0.5 **灵魂行者** 需单独规划天赋与精魂。

### 三大升华方向

| 升华 | 玩法类型 | 核心机制 |
|------|------|----------|
| **亚马逊** | 精英战矛战士 | 超凡战技、矛术强化、传统近战 |
| **仪祭师** | 仪式献祭 | 阿兹莫里黑暗仪式、牺牲换力量 |
| **灵魂行者** | 魂灵伙伴 | 圣洁鬼灵、驯兽、万灵意志（0.5 新） |

---

## 装备体系

### 防御思路

- **格挡 + 闪避** 可并行投资；格挡未触发时由生命与抗性承伤。
- 盾牌（若 BD 使用）提升格挡率；部分攻击技能要求双手武器。

### 武器与词缀

| 阶段 | 常见词缀 |
|------|------|
| 剧情 | 战矛物理点伤、攻速 |
| 过渡 | +矛/战矛技能等级、格挡相关 |
| 异界 | 生命、闪避、格挡率/格挡回复、四抗 |

### 饰品

- 手套：攻速；鞋：移速 30%+。
- 灵魂行者 BD 额外关注：**精魂、召唤/伙伴等级**（若适用）。

---

## 机制说明

1. **格挡** 有上限；未触发时由生命与抗性承伤。
2. 双手矛与盾/副手通常不可兼得，依 BD 选定武器方案。
3. 灵魂行者机制文档仍在迭代中。
4. 剧情装备奖励：战矛面板、缺抗补抗。

## 通用开荒提示（国服）

1. 技能宝石：每升一级检查任务奖励与商人。
2. 装备词缀：生命、四抗、鞋 **30%+ 移动速度**、格挡/闪避。
3. 0.5 赛季：法罗任务；≤55 级护甲可附加 **符文结界**。
4. 属性不足：城镇 **兜帽人** 洗点。
5. 国服组队稀有掉落有独立系数；见版本备忘与 FAQ。
