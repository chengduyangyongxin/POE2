## 文档结构

每个职业目录：

```
{职业}/
├── README.md      # 职业机制：技能 / 天赋 / 装备 / 机制说明

├── BD02-*.md      # 另一套构筑
└── BD03-*.md      # 变体或过渡构筑
```

每篇 BD 含：概述、升华、技能、天赋、装备、剧情奖励、异界过渡、参考链接。

## 国服 0.5 机制摘要

1. 1 级可访问市集；剧情装备奖励见各 BD 第六节。
2. 法罗任务解锁符文锻造；≤55 级护甲可附加 **符文结界**。
3. 鞋类常见词缀：**30%+ 移动速度**；防御向堆叠生命与四抗。
4. 精魂占用紧张时，主技能四连或五连通常先于捷光环等常驻技能启用。
5. 文档状态见各 README「文档状态」表。
