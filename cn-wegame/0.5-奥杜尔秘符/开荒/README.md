## 通用提示（国服 0.5）

1. 1 级即可市集购买；剧情装备奖励见各 BD 第六节。
2. 尽早做法罗任务，≤55 级护甲做 **符文结界**。
3. 鞋优先 **30%+ 移动速度**；生命 + 四抗打底。
4. 精魂紧张时先保主技能四/五连。
5. 所有 BD 状态均为 **待实战校对**，欢迎国服反馈修正。
