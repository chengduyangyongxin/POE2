# 德鲁伊 · 0.5 开荒攻略索引

> 国服 · 奥杜尔秘符赛季 · 开荒向  
> 技能译名以 [POE2DB 中文](https://poe2db.tw/cn/) 为准

---

## 职业基础

| 项目 | 说明 |
|------|------|
| 主属性 | **力量** + **智慧** |
| 主武器 | **魔符**（Druid 专属武器类型） |
| 防御倾向 | **生命** + 护甲/闪避 hybrid；部分 BD 用能量护盾 |
| 职业特性 | 可强化 **自身及召唤物**；天赋树 **攻击与法术伤害** 可互通 |
| 可选升华 | **萨满**、**神谕者** |

德鲁伊在近战、法术、召唤之间切换，是 0.5 国服 **新职业** 之一，机制丰富但学习成本高。

---

## 技能体系

### 武器与技能类型

- **魔符** 为专属武器，技能分 **野性攻击、自然法术、兽伴** 等多条线路。
- 可在 **近身搏杀** 与 **远程自然法术** 间切换，类似「战斗德鲁伊」。
- 召唤 **野兽/藤蔓** 等可与自身输出并行。

### 精魂与光环

- 兽伴、增益、自然光环可能占精魂，与女巫类似需 **精打细算**。
- 开荒：主攻击/主法术 → 关键召唤 → 光环。

### 技能连接要点

| 概念 | 作用 |
|------|------|
| **野性技能** | 近战物理/元素混合 |
| **自然法术** | 藤蔓、风暴、大地伤害 |
| **兽伴/召唤** | 脱手 DPS，吃召唤等级 |
| **变形**（部分 BD） | 形态切换换技能组 |
| **怒火/共鸣**（萨满） | 叠层爆发 |

---

## 天赋与升华

### 天赋树特点

- 力量与智慧双起点区域，**攻击% 与 法术% 可同树受益** 是德鲁伊特色。
- 可同时服务「挥砍 + 法术」混合 BD，但专精一点通常更强。
- 双属性穿装备压力大，注意 **博学** 与装备补属性。

### 两大升华方向

| 升华 | 定位 | 核心机制 |
|------|------|----------|
| **萨满** | 怒火元素 | 狂暴、自然共鸣、元素碾压 |
| **神谕者** | 命运之力 | 窥见未定之力、专属命运机制 |

**开荒建议**：社区资料尚少，建议关注国服实战 BD；萨满偏进攻，神谕者偏机制与成长。

## 装备体系

### 防御思路

- 混合职业建议 **生命 + 一种主防御**（护甲或闪避），勿全堆输出。
- 召唤向 BD 与女巫类似，找 **+召唤等级** 装备。
- 0.5 **符文结界** 对近战德鲁伊同样有效。

### 武器与词缀

| 阶段 | 优先 |
|------|------|
| 剧情 | 魔符物理/元素点伤、攻速或施法速度（看 BD） |
| 过渡 | +野性/自然/召唤技能等级 |
| 异界 | 生命、四抗、双属性；混合 BD 攻防词缀都要 |

### 饰品

- 戒指/项链补 **力量、智慧** 常见。
- 鞋：移速 30%+；召唤流追 **+召唤等级** 头/链。

---

## 开荒注意事项

1. **新职业** BD 与物价仍在沉淀，开荒成本波动大。
2. 攻击/法术双修听起来美，剧情阶段 **精魂与天赋点** 往往不够双线开花，建议先定主轴。
3. 魔符武器市集较少，剧情阶段多靠 **掉落与任务奖励**。
4. 兽伴死亡有召回成本，注意生存与拉怪节奏。
5. 剧情装备奖励：魔符面板、双属性、缺抗补抗。

## 通用开荒提示（国服）

1. 技能宝石：每升一级检查任务奖励与商人，优先凑齐主技能孔位。
2. 装备：优先 **生命、四抗、移动速度（鞋 30%+）**。
3. 0.5 赛季：尽早做法罗任务，低等级护甲加 **符文结界**。
4. 属性不够：城镇 **兜帽人** 洗点；力量/智慧双缺优先装备。
5. 国服组队掉落削弱，独狼或固定队往往更划算。
