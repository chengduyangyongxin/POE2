# BD01 · 神谕者 · 纠缠植物开荒

> 国服 0.5 奥杜尔秘符 · 剧情至异界初期  
> 难度：★★☆☆☆ · 造价：低 · **德鲁伊 T0 开荒**

---

## 一、BD 概述

与魔巫植物流相同技能轴：**纠缠** + **鞭笞藤蔓** + **雷霆风暴**，神谕者 **强制结局**（必暴）联动 **暴击时施法** 自动浇水。德鲁伊最省心开荒方案。

| 优势 | 劣势 |
|------|------|
| 操作极少 | 新职业 BD 资料仍在沉淀 |
| 暴击护盾生存 | 法术等级依赖高 |
| 与魔巫植物技能通用 | 魔符武器市集较少 |

---

## 二、升华：神谕者

| 顺序 | 升华节点 | 作用 |
|------|----------|------|
| 1 | 命运/暴击小点 | 基础 |
| 2 | **强制结局** | 必暴 → CoC |
| 3 | 窥见未定之力 | 伤害成长 |
| 4+ | ES、法术等级 | 后期 |

---

## 三、技能

同 [魔巫 BD01](../魔巫/BD01-瓦拉煞门徒-纠缠植物开荒.md)：

```
纠缠 — 加速生长 I — 分叉裂隙 I — 审议
雷霆风暴 — 范围扩大 I
```

精魂够：

```
暴击时施法 — 雷霆风暴 — 活体闪电 II
```

### 循环

- 纠缠 → 藤蔓 → 风暴；CoC 后全自动。
