# BD02 · 武圣 · 旋风击开荒

> 国服 0.5 奥杜尔秘符 · 0.5 新升华徒手流  
> 难度：★★★★☆ · 造价：低

---

## 一、BD 概述

1–40 级同 [BD01 冰川之刺](./BD01-祈求者-冰川之刺开荒.md)，41 级转 **旋风击** + **空心形态**。武圣 **石拳之道、空心专注** 是 0.5 行者热门终局方向，开荒需一定等级支撑。

| 优势 | 劣势 |
|------|------|
| 徒手旋风清图极强 | 41 级前同 BD01 |
| 武圣 0.5 新机制 | 学习曲线陡 |
| 连击爆发高 | 对天赋点需求大 |

### 41 级前

### 41 级+

