# 行者 · 0.5 开荒攻略索引

> 国服 · 奥杜尔秘符赛季 · 开荒向  
> 技能译名以 [POE2DB 中文](https://poe2db.tw/cn/) 为准

---

## 职业基础

| 项目 | 说明 |
|------|------|
| 主属性 | **敏捷** + **智慧**（双属性） |
| 主武器 | **徒手**、**棍/节杖** 等 |
| 防御倾向 | **闪避** + 生命；部分 BD 用能量护盾 |
| 职业特性 | **训练有素**，善于 **闪避** 袭来的攻击 |
| 可选升华 | **祈求者**、**夏乌拉追随者**、**武圣** |

近战职业；主属性 **敏捷** 与 **智慧**；技能含徒手、棍棒与念动法术。0.5 版本含 **武圣** 升华（徒手、幻象机制）。

---

## 技能体系

### 武器与技能类型

- **徒手** 与 **棍棒** 技能线不同，武圣多走徒手，祈求者多走元素棍/念动。
- 大量技能依赖 **连击、蓄力、突进** 机制。
- 部分技能 **无需手持武器**（徒手），装备栏可更灵活。

### 精魂与光环

- 祈求者元素化身、夏乌拉追随者混沌技能可能占精魂。
- 武圣幻象机制与精魂占用因 BD 而异；精魂按主技能连接分配。

### 技能连接要点

| 概念 | 作用 |
|------|------|
| **突进 / 翻滚** | 进出战场的核心 |
| **连击 / 蓄力** | 叠层增伤，武圣常见 |
| **元素附着**（祈求者） | 冰火雷联动，念动施法 |
| **混沌裂隙**（夏乌拉） | 混沌伤、吸收/造成混合 |
| **幻象 / 幻影之钟**（武圣） | 分身输出、时空机制 |

---

## 天赋与升华

### 天赋树特点

- 起点在敏捷与智慧交界，可点 **闪避、攻速、元素、生命、能量护盾**。
- 双属性需求：装备与天赋需同时照顾 **敏捷、智慧**。
- 武圣与祈求者天赋走向差异大，洗点成本需注意。

### 三大升华方向

| 升华 | 玩法类型 | 核心机制 |
|------|------|----------|
| **祈求者** | 元素宗师 | 元素化身、念动施法、精神超脱 |
| **夏乌拉追随者** | 混沌武僧 | 黑暗掌控、裂隙力量、元素换混沌 |
| **武圣** | 徒手幻象 | 人形兵器、幻影之钟、连击爆发（0.5 新） |

---

## 装备体系

### 防御思路

- **闪避** 为主防御层；突进技能用于位移。
- 祈求者部分 BD 可堆 **能量护盾**；夏乌拉可能有 **吸收伤害** 机制。
- 徒手 BD 注意 **胸甲/手套** 是否限制徒手技能。

### 武器与词缀

| 阶段 | 常见词缀 |
|------|------|
| 剧情 | 棍棒点伤/攻速，或徒手向词缀（无武器要求时看护甲） |
| 过渡 | +徒手/棍技能等级、攻速、元素点伤（祈求者） |
| 异界 | 生命、闪避、四抗；双属性装备 |

### 饰品

- 项链/戒指补 **敏捷、智慧** 是常见需求。
- 鞋：**30%+ 移速** 为近战突进职业常见目标词缀。

---

## 机制说明

1. 装备与天赋需同时满足 **敏捷、智慧** 需求；可用 **博学** 与任务奖励补属性。
2. 徒手 BD 需确认技能对 **武器** 与 **护甲基底** 的限制。
3. 连击类技能中断会降低伤害输出。
4. 剧情装备奖励：敏捷/智慧属性、武器面板。

## 通用开荒提示（国服）

1. 技能宝石：每升一级检查任务奖励与商人。
2. 装备词缀：生命、四抗、鞋 **30%+ 移动速度**、闪避。
3. 0.5 赛季：法罗任务；≤55 级护甲可附加 **符文结界**。
4. 属性不足：城镇 **兜帽人** 洗点；敏捷/智慧由装备补充。
5. 国服组队稀有掉落有独立系数；见版本备忘与 FAQ。
