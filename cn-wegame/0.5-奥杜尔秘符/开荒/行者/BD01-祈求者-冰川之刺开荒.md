# BD01 · 祈求者 · 冰川之刺开荒

> 国服 0.5 奥杜尔秘符 · 三升华通用剧情 BD  
> 难度：★★★☆☆ · 造价：低 · **行者 T0 开荒**

---

## 一、BD 概述

**冰凌座** + **冰川之刺** 念动冰爆，祈求者元素化身。Act1 末点 **空心掌** 后转徒手，清图与 BOSS 均衡。武圣玩家 1–40 级可走同一技能轴。

| 优势 | 劣势 |
|------|------|
| 冻结控场、伤害稳定 | 双属性装备压力 |
| 三升华前期通用 | Act4 前偏护甲需换闪避 ES |
| 徒手后机动高 | 念动机制需熟悉 |

## 三、技能

### 3.1 学习顺序

1. **移风岚** / 节杖过渡（1–12 级）
2. **冰凌座** + **冰川之刺**（核心）
3. **寒霜爆**、**冰霜之捷**
4. Act1 末：**空心掌** 天赋（徒手）
5. BOSS：**风暴乱舞**（感电）+ **冰冻印记**

### 3.2 技能连接

```
冰川之刺 — 冰咬 I — 冰川化 I — 审议
```

```
冰凌座 — 范围扩大 I — 效能 I
```

### 3.3 循环

- 清图：冰凌座铺场 → 冰川之刺引爆。
- BOSS：寒霜爆 → 印记 → 冰川之刺连发。

- Act4+：**闪避、能量护盾、冰霜**
